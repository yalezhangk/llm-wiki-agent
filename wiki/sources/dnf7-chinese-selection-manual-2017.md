## Summary
这份中文 PDF 适合按 `2017-08-16` 的 DNF7 中文工程选型手册层处理，而不是简单并入已入库的英文 `2017` 手册或 `2025-04-10` 中文数字化样本。它保留了与较早英文工程资料连续的技术骨架，继续围绕 `36kV / 40.5kV`、`HVX / FP`、四隔室、`LSC2B / PM` 和 `31.5kA` 级内部电弧能力展开，但明显更偏中国市场正式工程资料。它真正新增的价值，是把 DNF7 的中文标准口径、`KYN28 / KYN61` 两种结构、一次方案页和土建安装页完整展开，构成 `2017` 时点更丰富的本地化选型层。

## Key Claims
- 文档封面将 DNF7 定义为“中压户内开关设备 / 户内交流金属封闭开关设备”，元数据日期为 `2017-08-16`。
- 概述页把 DNF7 写成适用于 `40.5kV(GB)` 与 `36kV(IEC)` 的空气绝缘中压开关柜，额定电流最高 `3150A`，并明确支持 `HVX` 与 `FP` 同电流参数下互换。
- 手册明确给出两种主流结构：中置式 `KYN28` 和落地式 `KYN61`。
- 资料按中文工程手册方式展开了产品特征、执行标准、结构描述、整柜参数、`HVX`/`FP`/接地开关参数、一次方案、布置与安装、外形尺寸与重量。
- 相比 `2017-08-21` 英文手册，这份中文版本更突出五防闭锁、关门操作、释压通道、电缆室空间、`1200mm / 1400mm` 柜宽、基础框架和配电室布置。
