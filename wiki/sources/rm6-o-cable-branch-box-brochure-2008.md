## Summary
这份 `8` 页中文折页把 [[RM6O]] 明确写成施耐德 `3~24kV` 的 `SF6` 绝缘电缆分支箱，是当前知识库里 RM6 路线最早可确认的分支箱产品资料。它不是环网柜整柜目录，而是基于 `RM6-I` 与 `RM6-O` 模块、面向电缆分支与扩展出线场景的独立产品简介。

## Key Claims
- 封面写有 `RM6-O系列电缆分支箱` 与 `3至24kV`，PDF 元数据创建时间为 `2008-12-10`，尾标为 `SCDOC723-MV / 2008.12`。
- 文档把 RM6-O 定位为替代配网架空线大量分支接线的一种经济、可靠、维护方便方案，并明确写为 `SF6` 绝缘电缆分支箱。
- 正常环境条件按 `IEC60694` 的户内开关装置口径给出：环境温度 `-40℃~40℃`，海拔 `≤1000m`，高于 `1000m` 需咨询。
- 电气参数覆盖 `12kV` 与 `24kV`：额定电流 `630A`，`12kV` 额定开断电流 `20/25kA`、短时耐受 `25kA`，`24kV` 额定开断电流 `20kA`、短时耐受 `20kA`，关合电流峰值分别到 `50/63kA` 与 `50kA`。
- 结构上采用 `1~4` 个集成化紧凑功能单元和 `1` 个不锈钢 `SF6` 金属封闭气室，将 `RM6-I` 与 `RM6-O` 模块永久密封在额定压力 `1.2Bar` 的气室内。
- 文档明确写出运行设计寿命超过 `30` 年，所有带电部件免维护，并满足 IEC 对 `sealed pressure system` 的定义。
- 产品可做一进二出、三出或更多出线；出线较多时可通过主母线扩展附件扩展，但扩展后主母线额定电流不能超过 `630A`。
- 每路进出线均可接双电缆，每路均配置电压指示器；还可选装 `Flair 22D` 电缆故障指示器、单孔或三孔底板。
- 每路进线可选装电动操作和辅助接点，预留配网自动化接口；电缆出线与负荷开关之间可选装机械联锁以保障运行和检修安全。
- 安装页补充了不可扩展、右扩展和双扩展外形尺寸，地面安装、靠墙安装、`260/520mm` 升高座，以及考虑内燃弧能力时向后部或下部泄压的土建边界。
