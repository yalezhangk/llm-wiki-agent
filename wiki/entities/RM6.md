## Summary
[[RM6]] 是 [[施耐德电气]] 的 `12~24kV` [[气体绝缘开关柜]] / 环网柜产品系列，面向中压二次配电场景。当前 wiki 中它除了 `2018` 演示稿、`2022.02` [[RM6SeT]] 目录和 `2025.02` [[RM6SeT]] 修订目录之外，还补入了一份更偏运维手册口径的 [[RM6S]] 说明书：这说明 RM6 路线既有面向产品宣传与工程目录的版本链，也有保留较早 `12kV` 平台操作维护边界的现场手册体系。

## Key Details
- 演示稿称 RM6 在全球 `50` 多个国家已有超过 `1,700,000` 个功能单元应用，并以“20 多年不断发展与调整”强调成熟度。
- 核心电气口径覆盖 `12kV` 与 `24kV`；`I/B/O` 功能单元额定电流 `630A`，`Q/D` 功能单元额定电流 `200A`。
- 绝缘水平包括 `12kV` 对地工频 `42kV`、断口 `48kV`、冲击 `95/110kV`，以及 `24kV` 对地工频 `65kV`、断口 `79kV`、冲击 `125/145kV`。
- 防护等级为 `IP4X`，短时耐受能力为 `12kV 20kA 4s`、`24kV 20kA 3s`，内部燃弧能力为 `20kA 1s`。
- 文档把 RM6 描述为全绝缘、紧凑型方案，强调带电部件位于不锈钢 `SF6` 气箱内、可触摸、年泄漏率 `<0.01%`，并给出 `IPX7 20kV` 水下通电 `30` 分钟口径。
- 功能单元体系包括 `I / IC` 负荷开关与联络开关、`B / BC / D` 断路器单元、`Q` 熔断器组合电器、`Mt` 中压计量和 `O` 电缆提升。
- `B / BC / D` 断路器单元覆盖线路保护、母联和变压器馈线保护，强调自供电保护、远程控制和“两步从合闸位到接地位”的简化操作。
- 保护继电器和保护装置路线包括 `VIP45`、`VIP400`、`VIP410` 和 `Sepam`，其中 `VIP410` 支持双电源供电、灵敏接地保护、`Modbus RS485` 通信和远程控制。
- 柜型体系给出 `58` 种标准柜型，包含紧凑不可扩展、紧凑可扩展、左扩、右扩、双扩和 `2/3` 功能自由组合柜；资料称自由组合柜型配置数量可达 `600`。
- 文档还单独列出针对轨道交通的不可扩展共箱方案 `NE-IQI+T`、`NE-QIQI+T`、`NE-IIQI+T` 和 `NE-IQIQI+T`。
- 自动化与附件体系覆盖 `VD23` 电压检测、`VPIS/VDS`、`Flair 22D/23DV` 故障指示器、电流表、电操机构、程序锁、电缆室门连锁、电磁锁、压力开关和 `ATS` 双电源自动投切。
- `ATS` 方案强调双重机械/电气联锁、手动优先、故障闭锁、`RS485` 与以太网通信，以及小于 `10s` 的自动投切目标。
- `Smart RMU` 章节则把 RM6 延伸到 `Easergy T300`、低功耗电压/电流传感器、温湿度传感器、`TH110`、UHF 局放、`wifi`、`Webserver` 和移动运维。
- `2022.02` 的 [[RM6SeT]] 目录把 RM6 路线进一步工程化：给出主回路 `40` 年寿命、`3mm 304` 不锈钢气箱、气箱 `IP67`、电缆室 `IP43`、`IK07`、`1~6` 功能共箱、`A-link` 扩展、`C/F/V/D/PT` 功能体系，以及 `Easergy P1`、`PD110`、`TH110-R`、`CL110`、`SMD` 和 `Easergy T300` 组成的数字化状态监测与自动化能力。
- 相比 `2018` 演示稿，`RM6 SeT` 目录还明确了若干边界：`PT` 和 `M` 柜不能共箱，只能通过可扩展柜连接；内部电弧 `A FLR 20kA 1s` 为可选项；`V` 真空断路器可支持自动重合闸顺序 `O - 0.3s - CO - 3 min - CO`。
- `2025.02` 的 [[RM6SeT]] 修订目录继续沿用 RM6 SeT 主结构和数字化栈，但把 `Cr / Vr / M` 模块显式写进目录，并在部分 `12kV` 参数表中补入 `25kA` 短时耐受、`63kA` 峰值耐受等更高能力选项。
- [[RM6S]] 手册则从另一侧补充 RM6 路线的历史边界：对象是 `12kV` `SF6` 气体绝缘金属封闭开关设备，功能单元代号为 `OE / I / Q / B / IC`，并围绕 `Flair2XD`、`VPIS-V2`、`T200I`、`VD23`、电缆试验、`0.03 MPa @ 20℃` 充气压力和压力表绿色工作区给出操作维护要求。

- `2020.07` 的 [[RMAirSeT]] 资料说明 RM 路线还存在一条环保气体分支：采用 `GWP < 1` 环保气体绝缘、真空灭弧和 `SVI` 并联真空开断，保留 `IP67` 气箱、`1~6` 功能单元自由组合、`NE/LE/RE/DE` 扩展与 `C/F/V/B` 功能体系。
- 相比 [[RM6SeT]]，[[RMAirSeT]] 更像 `12kV`、`20kA`、环保气体 + 真空灭弧的平行 RMU 路线，而不是简单改名或完整替代；其数字化配置继续围绕 `Easergy T300`、`PMBOX`、`PD110`、`TH110-R`、`VT/LPVT` 和 `CT/LVCT` 展开。
