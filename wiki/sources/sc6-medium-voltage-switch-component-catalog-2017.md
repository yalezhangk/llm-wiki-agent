## Summary
这份 PDF 的文件名写作“AS12中压开关 产品目录”，但正文真实题名是 `SC6`，封面写明“中压开关元件 SC6 可靠核‘芯’ 6-12kV 630A 产品目录”，尾页文件号为 `SCDOC1158-MV`，时间口径为 `2017.09`。它不是完整开关柜目录，而是施耐德面向空气绝缘柜体集成场景提供的 `12kV` 中压开关元件收敛版资料，保留了与更早 [[AS12AS24]] 路线的连续性，同时把功能和电压口径收敛到 `SIC / SQC` 与 `12kV`。

## Key Claims
- 文件名使用 `AS12`，但正文产品名明确为 `SC6`，应按正文题名建档，同时保留其与 [[AS12AS24]] 的版本连续性。
- 封面给出 `SC6`、`6-12kV`、`630A` 和“中压开关元件”定位，尾页写明文件号 `SCDOC1158-MV`、版本时间 `2017.09`。
- 产品继续被定义为“空气绝缘开关柜的核心元件”，柜体仍由国内开关柜制造商完成，施耐德提供核心开关本体与附件体系。
- 功能系列收敛为 `SIC` 和 `SQC`：分别适用于负荷开关进出线柜/连接柜，以及熔断器开关组合柜。
- 参数口径集中在 `12kV`、`630A`、`20kA/4s`，峰值耐受电流 `50kA`，短路关合电流 `50kA`。
- 资料还给出 `10A` 电缆充电电流开断、`94.5A` 接地故障电流开断、`31.5A` 的 `5%` 额定有功负载开断，以及 `1750A` 最大转移电流和 `630A` 闭环开断能力。
- 绝缘水平写为工频耐压 `42/48kV`、雷电冲击耐压 `75/85kV`。
- 技术页继续采用 `SF6` 作为绝缘和灭弧介质，气室相对压力 `0.4 bar`，年泄漏率 `<0.05%`。
- 机械寿命口径为主开关 `5000` 次、接地开关 `3000` 次，电气寿命等级写为 `E3`。
- 安全与可视化边界包括三位置开关、天然联锁、观察窗、同轴指示、柜门/接地/主开关联锁、柜后泄压，以及可选 `VPIS-V2`、`SF6` 压力表和带接点输出压力开关。
- 附件体系包括 `Fusarc CF`、`Easergy Amp 21D`、`Easergy Flair 22D`、`Talus 200I` 和钥匙闭锁。
- 技术页正文仍出现“`AS12` 开关使用 `SF6` 气体作为绝缘和灭弧介质”的表述，说明这份 `SC6` 目录很可能延续自更早的 `AS12` 平台文案与结构。

## Contradictions
- 与已入库 [[AS12AS24]] 资料不构成冲突，更像 `2010 -> 2017` 的版本演进：前者覆盖 `12/24kV` 与 `IS / QS / DS / CS / ES`，后者收敛到 `12kV` 与 `SIC / SQC`。
- 它不是完整整柜目录，不能拿 `SC6` 的单型号元件参数去替换 [[SM6]]、[[RM6]]、[[FBX]] 或 [[PIX]] 等整柜产品线口径。
- 虽然处于 [[空气绝缘开关设备]] 场景，但开关本体仍使用 `SF6` 密闭压力系统，因此应继续理解为“空气绝缘柜体中的 `SF6` 开关核心”，而不是纯无气体绝缘路线。
