## Summary
这份 `SCDOC1738 / 2018.10` 中文目录把 [[Premset]] 从单纯安装对象补充为完整的 `12kV` 中压开关柜产品线。它的核心定位不是传统 [[气体绝缘开关柜]] 或典型 [[空气绝缘开关设备]]，而是“[[屏蔽式固体绝缘]]系统”：以相间绝缘与屏蔽来降低内部故障风险，同时用 `375mm` 标准宽度、`IP67` 主回路、`PM` 金属隔板、`LSC2A` 连续性和 `25kA x 1s A-FLR` 内燃弧防护构成紧凑型模块化真空开关设备方案。

## Key Claims
- [[Premset]] 被目录定义为 `12kV` 紧凑型模块化真空开关设备，封面副标题为“屏蔽式全绝缘系统”。
- 主要功能单元包括进线/馈线、母线分段、母线提升、线路保护、母线 `PT` 柜，以及计量和测量单元；典型柜型包括 `D06H`、`D12H`、`I06T`、`I12H`、`G06`、`G12`、`M06S`、`M06A`、`M12S`、`M12A`。
- 基础额定参数为 `12kV`、`50/60Hz`、母线电流 `630A / 1250A`，额定雷电冲击耐受电压 `75kV`，隔离断口间 `85kV`。
- 额定短时耐受电流和持续时间给出 `20kA x 4s`、`25kA x 3s`、`25kA x 4s`、`31.5kA x 4s` 多种口径。
- 断路器开断能力为 `E2` 级：`12kV 25kA 50次` 或 `12kV 31.5kA 30次`；容性电流开断能力为 `C2` 级 `12kV 25A`。
- 机械寿命口径包括断路器 `M2级 10000次`、负荷开关 `10000次`、接地开关 `3000次`。
- 目录强调所有具备 [[屏蔽式固体绝缘]] 的开关、断路器和计量单元宽度均为 `375mm`（`630A`），空气绝缘计量单元和进线计量单元为 `750mm` 宽；系统深度 `1135mm`，电缆联接高度 `700mm`。
- 根据 `IEC 62271-200` 第2版和 `GB 3906-2006`，Premset 每个隔室都经过内燃弧测试，给出 `25kA x 1s A-FLR` 的四周防护口径，并记录 `20kA` 和 `25kA` 内燃弧型式试验。
- 运行条件页给出 `PM` 金属隔板、`LSC2A` 运行连续性、外壳 `IP3X`、隔室之间 `IP2X`、主回路和所有高压部件 `IP67`（`M06A` 计量柜除外）、`IK07`。
- 环境适应性口径包括 `5000m` 以下无需额外预防措施、`-25℃ ~ +40℃` 常规运行、可选到 `+65℃`，以及“浸水后可连续服务 `48小时`（仅限固体绝缘柜型）”。
- 联接章节强调带 [[屏蔽式固体绝缘]] 技术（`2SIS`）的电缆与母线接口“完全免维护和免维修”，并通过通用扁平联接系统支持前下、后上、后下等多种布置。
- 电缆测试章节说明无需操作主开关或拆除电缆接头即可直接测试主回路电缆，并提供专用测试设备与五防联锁逻辑。
- 技术附录给出底部/顶部泄压两种内燃弧布置方式，以及 `≥100mm` 靠墙距离、最小电缆沟深度和柜体固定要求。
