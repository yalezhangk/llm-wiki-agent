## Summary
[[RMAirSeT]] 是 [[施耐德电气]] RM 路线中采用环保气体绝缘和真空灭弧的 `12kV` [[气体绝缘开关柜]] / 环网柜分支。当前资料更适合把它理解为与 [[RM6]] / [[RM6SeT]] 并行的环保气体 RMU 平台，而不是 [[GMAirSeT]] 那种一次配电平台，也不是 RM6 SeT 的简单改名版本。

## Key Details
- 封面核心卖点包括 `GWP < 1`、全生命周期碳足迹减低、不燃。
- 结构上保留 RMU 典型组合逻辑：最多 `6` 个功能单元，自由组合，并以 `NE / LE / RE / DE` 表示不扩展、左扩展、右扩展和双扩展。
- 功能单元写为 `C` 三工位负荷开关、`F` 负荷开关-熔断器组合电器、`V` 上隔离真空断路器、`B` 下隔离真空断路器，并提到 `PT`、计量柜和母联等扩展对象。
- 文档把 `SVI` 并联真空开断技术作为关键差异点之一，说明环保气体路线并未放弃真空开断叙事。
- 技术参数页给出 `12kV`、`20kA` 短路开断、`20kA / 4s` 短时耐受、`50kA` 峰值耐受、`A-FLR 20kA 1s` 和 `IP4X / IP67`。
- 气体年泄漏率为 `≤0.025%`，额定充气压力为 `0.4 bar`。
- 额定电流口径包括 `C` 单元 `630A`、`F` 单元 `125A`、`V/B` 单元 `630A`；`V/B` 机械寿命 `10000` 次，`C/F` 机械寿命 `5000` 次。
- 外形尺寸以 `420mm x 850mm x 2000mm` 级单元柜为主，深度可选 `780mm`，部分一二次融合/PT 柜宽度为 `600mm`。
- 使用环境边界为 `-25℃ ~ +40℃`、高湿度工况和常规海拔 `1000m`，更高海拔需咨询施耐德。
- 数字化配置直接列出 `Easergy T300`、`HMI`、`PMBOX`、`PD110`、`TH110-R`、`VT/LPVT`、`CT/LVCT` 和 `Modbus`。
- 文档强调实时感知、IT/OT 融合、资产预警和预测性资产管理，说明 [[RMAirSeT]] 不只是环保替代，也承担数字化运维入口角色。
