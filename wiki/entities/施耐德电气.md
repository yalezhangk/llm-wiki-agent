## Summary
[[施耐德电气]] 是这份资料中的厂商主体，被描述为全球能效管理和自动化领域的专家。文档用它来承接 [[EcoStruxure]] 架构、[[SmartHVX]] 智能中压断路器、[[PMBEasy]] 主动运维单元，以及 [[GHA]]、[[GMA]]、[[PIX]] 等中压开关柜产品和方案。

## Key Details
- 文档将施耐德定位为面向安全、可靠、高效和可持续能源管理的全球企业。
- 中国区介绍强调本地员工规模、工厂、物流中心、研发中心与办事处分布。
- [[SmartHVX]] 被放在施耐德中压配电和数字化运维能力的产品叙事中。

- `SCDOC-DVCAS-EN` 则把这条更早的环网柜 / RMU 图谱补成一份完整风电场专用目录：资料把 [[DVCAS]] 写成 `36kV` 风电场集电网二次配电开关柜，强调 `600mm` 门洞通过能力、`2000m` 海拔、`SF6 0.3bar` 密封气箱、`IP67` 高压隔室、`I / D / 0 / T` 模块体系，以及 `VIP-35`、`Flair`、`48Vdc` 电动机构和远方故障隔离恢复供电逻辑。
- `风能解决方案` 说明到 `2018.10`，施耐德已经把风电行业整理成四组系统级方案：风机系统、风场连接、并网电站和电力监控/视频安防，并把 [[FBX]]、[[FLUSARC]]、[[WSG]]、[[DNF7]]、[[Trihal]]、[[Premset]]、[[PLSCADA]] 和 [[iRIO]] 放进同一套风能叙事。
- `FLUSARC 40.5 kV` 中文目录说明到 `2016.07`，施耐德已经在中国市场以 `SCDOC1545` 形式发布独立的 `40.5kV` `SF6` 绝缘中压开关柜目录；公司页使用 `2015` 财年 `270亿欧元`、全球 `16万` 员工、中国 `26000` 员工、`26` 家工厂、`8` 个物流中心、`5` 个分公司和 `40` 个办事处的历史口径。
- `FOX中压开关柜核心元件产品目录` 说明到 `2016.03`，施耐德已在中国市场发布 `12kV` 以下 `SF6` 气体绝缘开关设备核心元件目录；它强调“施耐德提供核芯元件、合作伙伴生产电缆室和低压间”的合作分工，属于不同于完整整柜目录的集成路线。
