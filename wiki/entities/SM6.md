## Summary
[[SM6]] 是 [[施耐德电气]] 在中压二次配电环网柜范围内的一条独立产品线。当前 wiki 中它已经形成 `2012` 英文正式目录、`2017` 中文 `24kV` 正式目录、`2018.10` `SMD` 培训稿与 `2019-03-27` 中文产品/数字化演示稿四层资料链：`2012` 目录把它写成最高 `36 kV` 的模块化空气绝缘开关柜，`2017` 目录把中国市场正式口径收敛到 `24 kV` 以下并补齐选型/订货与自动化方案，`2019` 资料则把市场叙事聚焦到 `7.2~24kV` 环网柜和 `Smart SM6`，而 `2018` 培训稿进一步补齐 `SMD / TH110 / CL110` 的实施层。因此更适合把它理解为“空气绝缘柜体 + `SF6` 开关隔室 + 数字化监测层”的边界型环网柜平台。

## Key Details
- `2012` 英文正式目录的封面题名为 `SM6 Modular units Air insulated switchgear up to 36 kV`，PDF 元数据日期为 `2012-07-10`，说明 [[SM6]] 至少存在一个更早、更正式的最高 `36 kV` 目录口径。
- 这份目录把产品定义为采用 `SF6 or vacuum breaking technology` 的模块化柜型体系，并写有 `30 years life span`，同时声称全球安装超过 `1,100,000` 台 `SM6` 柜体。
- 目录明确列出 `IM / IMC / IMB`、`PM`、`QM / QMC / QMB`、`CRM / CVM`、`DM1 / DM2`、`DMV / DMVL`、`CM / CM2`、`GBC-A / GBC-B`、`NSM`、`GIM / GEM / GBM / GAM / TM / SM` 等完整功能单元谱系。
- `2012` 目录还给出更清晰的结构与安全口径：`PI` 绝缘隔板、`LSC2A`、`IP3X`、`VPIS`、前方闭门操作、联锁、`anti-reflex` 手柄，以及 `24 kV` 的 `A-FL / A-FLR` 与 `36 kV` 的 `A-FL` 内部电弧等级。
- 这份正式目录再次确认了 SM6 的关键边界：虽然封面写的是空气绝缘开关柜，但 active parts integrated in an `SF6-filled, sealed for life` unit，因此不应把它写成纯无气体绝缘路线。
- `2017` 中文 `24kV` 正式目录号为 `SCDOC064-MV`，封面写作“中压开关柜 SM6 悉心守护 享受无忧 6～24kV 产品目录”，把中国市场正式口径落到 `24 kV` 以下的二次变电站和工矿配电站应用。
- 这份目录在柜型上补齐 `IM / IMC / IMB / IMP / QM / QMC / CRM / DM1-A / DM1-D / DM1-S / DMV / CM / CM2 / GBC-A / GBC-B / NSM-C / GBM / GAM2 / TM` 等中文工程命名，并给出 `375-750 mm` 宽、`1600 mm` 高、`1030 mm` 深、前接线和 `IP4X` 的实际安装口径。
- `2017` 目录给出 `7.2 / 12 / 24 kV`、`630 / 1250 A`、`20 kA / 4s` 或 `25 kA / 1s`、`LSC2A`、`PI`、`IAC A-FLR 20 kA / 1s` 等正式中文参数，并把 `QM/QMC`、`DM1-A/DM1-D` 的 `12 kV 25 kA / 24 kV 20 kA` 分断能力写得更具体。
- 它还补上较早期自动化和监测层：`Sepam 20/40/80`、`VIP`、`Easergy T200 S`、`Flair 22D`、`Amp 21D` 和双电源自动转换设备，说明 `Smart SM6` 之前 SM6 已有一套更传统的配网自动化口径。
- 已入库资料是一份 `20` 页中文演示稿，封面口径为 `SM6 7.2~24kV 空气绝缘环网柜`。
- 环网柜分类页把 `SM6`、`RM6/FBX` 和 [[Premset]] 并列，说明施耐德在二次配电环网柜上至少同时保留空气绝缘、气体绝缘和固体绝缘三条不同叙事路线。
- 文档前半部分先用“金属封闭开关设备”页面解释防护、绝缘、载流、控制、隔离、保护、计量和监测功能，再把 SM6 放回中压二次配电网络语境。
- 进出线柜页给出 `IM 375`、`IM 500`、`NSM-C 1000` 和 `IMC 500` 等柜型线索，服务于双馈线、主进线带备用线或发电机场景。
- 负荷/隔离开关柜页说明开关间隔内装三工位负荷开关，且“充有六氟化硫（SF6）气体为绝缘介质”；母线间隔可左右扩展，操作机构间隔可选电动操作机构，电缆间隔为正面连接。
- 续页把该路线的结构卖点写成三位置设计、天然联锁、位置显示、传动机构直接可靠、`1.4 bar` 气包和 `0.01%` 泄漏率。
- 保护柜页给出 `QM375/500`、`QMC625`、`DM1-A750` 等柜型，并把保护对象分为 `SF1 / SFset` 断路器、`SF6` 灭弧和负荷开关熔断器组合电器。
- 断路器柜页同样强调开关间隔充 `SF6` 且符合密闭压力系统标准，可安装 `SFset / SF1` 断路器。
- 两种断路器对比页给出清晰分工：`SF1` 配 `Sepam` 综合保护、标准电流互感器、需要辅助电源，适用 `630A` 和 `1250A`；`SFset` 为真空断路器型，配专用传感元件和 `VIP400` 电子继电器，不需要辅助电源，适用 `630A`。
- 测量柜页列出 `CM2500`、`GBC-B750`、`GBC-A750`；专用柜页列出 `GAM23750`、`IMB 375`、`GBM 375`、`DM1-D 750` 等特殊用途柜型。
- `Smart SM6` 是这份资料的另一半核心价值：它把 `Easergy T300`、`SCADA / DMS`、`101/104`、Local HMI 和现场传感器接到同一配网监控架构。
- 监测对象重点落在主母排搭接点与电缆搭接处温升、电缆室局放，以及母线室/电缆室环境温湿度；对应器件分别写成 `TH110 / TH110-R / TH110-S`、`PD110` 和 `CL110`。
- `SMD and Sensors for SM6 training v2` 则把 `Smart SM6` 从“监测对象列表”推进到“实施与版本层”：`SMD` 架构可接 `zigbee / PT100 / DI / relays`，输出到 Local HMI、Nearby HMI、`Modbus TCP`、`Facility Expert` 告警链路和 `Facility Hero` 移动告警。
- 同一培训稿还补出 `SMD V3 -> V4 -> V4.1 -> V5` 的版本链，以及 `Thermal Connect App`、`NFC tag`、batch pairing、`Easergy P3/P5` 兼容等部署信息，说明 SM6 路线的数字化叙事不只停留在传感器命名层。
- 培训稿中的 `partial discharges sensor` 仅出现在 `2019` wireless sensors roadmap 中，因此当前更适合作为施耐德计划方向记录，而不是回写成当时 SM6 已正式具备的统一标配。
- 因此，[[SM6]] 与 [[RM6]] / [[FBX]] 的差别不只是介质命名：RM6/FBX 更强调紧凑 `SF6` RMU 与目录化参数，而 SM6 的资料链更突出从 `36 kV` 模块化正式目录，到 `24 kV` 中文正式目录，再到中国市场柜型谱系和 `Smart SM6` 在线监测扩展的连续演进。
