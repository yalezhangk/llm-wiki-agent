## Summary
这份英文 PDF 是一份更早期的 [[DNF7]] 技术说明/选型手册，封面标题为 `DNF7 Metal-Clad Switchgear Air Insulated Switchgear Technical Instruction`，封底印有 `ONEXXXX 2010.12`，而 PDF 元数据 `CreationDate` 为 `2011-01-04`、`ModDate` 为 `2016-03-09`。它不是现有 `2025-04-10` 中文样本的重复件，而是 DNF7 在更早阶段的英文工程口径：重点落在 `36kV / 40.5kV` 系统适用范围、四隔室结构、`HVX` 与 `FP` 互换、`Middle Rolling / Floor Rolling` 两种抽出方式，以及一次方案和基础尺寸。它同时保留了带时间边界的早期公司介绍与市场口径，因此更适合用来补 DNF7 的版本链，而不是覆盖后续数字化样本中的较新叙事。

## Key Claims
- [[DNF7]] 被定义为面向户内应用的 air insulated metal-clad switchgear，适用于 `36kV` 和 `40.5kV` 系统。
- 早期英文手册明确 DNF7 采用四隔室结构：busbar、circuit breaker、cable 和 low voltage compartment，并按 `LSC2B` 与 `PM partition class` 组织。
- 文档强调可配置施耐德可抽出式 `HVX` 真空断路器与 `FP` 气体绝缘断路器，且相同额定值下 switching technology 可互换。
- 设备提供 `Middle Rolling` 与 `Floor Rolling` 两种 withdrawable 形式，并给出进线/馈线、计量/测量、母联/母线上升等一次方案页。
- 技术数据页给出 `36kV / 40.5kV`、`1250~3150A`、`AFLR 31.5kA (1s)`、外壳 `IP4X`、可选 `IP41`、标准尺寸 `1200(1400) x 2505 x 2400 mm` 和最大重量 `1800kg` 等较早期工程口径。
