## Summary
[[气体绝缘开关柜]] 是以气体介质承担主要绝缘功能的中压开关柜类别。[[GMA]] 文档中的气体绝缘路线以 SF6 密封气室、固体绝缘母线、金属封闭结构和模块化扩展为核心；[[GMAirSeT]] 则保留相近结构但采用干燥空气/环保气体，以回应低 GWP、碳中和和寿命期末免气体处理诉求。[[GHA]]、[[WSG]] 和 [[WIG]] 资料共同补充了更高电压等级或专用场景的气体绝缘金属封闭路线。

## Key Details
- GMA 的所有带电中压元件置于密封充气隔室内，母线采用固体绝缘技术。
- GM AirSeT 同样把带电中压元件置于密封充气隔室内，但绝缘介质为干燥空气/环保气体。
- 密封气室使中压元件不易受腐蚀性气体、污垢、灰尘和小动物影响。
- GMA 在安装和维护时无需任何气体处理，使用寿命结束时按回收数据表处理可循环材料。
- GM AirSeT 进一步强调寿命期末无需额外气体处理，避免 SF6 类温室气体处置问题。
- GMA 的充气隔室防护等级为 IP67。
- 每个充气隔室可通过温度补偿压力表监测，压力表有正常、报警和闭锁区域指示。
- GMA Air 数字化实施材料进一步把 [[气室监测]] 拆成气体密度、气体温度和气体湿度监测。
- 英文 GMA 目录把 gas-filled compartments 描述为 sealed pressure system，强调整个寿命期无需检查或补充绝缘气体。
- GHA 被描述为最高 40.5kV 的气体绝缘开关柜，支持单母线和双母线方案，采用真空断路器。
- GHA 2018 目录给出 12/17.5/24/36/38/40.5kV、40kA 短时电流、100kA 峰值耐受电流、母线/支路 2500A 的总额定参数，母线更高值可按需到 4000A。
- GHA 的 gas-filled compartments 同样采用 sealed pressure system 口径，资料明确引用 IEC/EN 62271-1；旧演示稿还给出泄漏率 `<< 0.1 % p.a.`。
- GHA 目录明确气室由铬镍钢制成，整寿命期免维护、无需补气，且安装、扩展、更换和拆除时不需要现场气体处理。
- GHA 的 [[BLink]] 使柜体安装、扩展、拆卸和柜列内移动可以避免现场气体处理。
- GHA 的 [[IDIS]] 把每个气室的气体密度状态转换为交通灯式就地显示和远程信号。
- GHA 含 SF6 气体，目录强调通过自密封阀回收 SF6，并给出不含包装按重量计 90% 的潜在可回收率。
- WS-G 企业标准将设备定义为 12kV~40.5kV、50Hz 的户内交流气体绝缘金属封闭开关设备，适用于单母线、双母线或母线分段系统。
- WS-G 的母线隔室和断路器隔室均充 SF6，电缆隔室为可触及隔室，其他隔室为不可触及隔室。
- WS-G 标准要求每个充气隔室设置带温度补偿的 SF6 密度监视装置，具备低气压报警和最低功能压力报警。
- WS-G 标准给出 SF6 年漏气率 ≤0.1%、气室湿度 250μL/L 和充气箱 IP65 等约束。
- WS-G 2024 版标准把特殊使用条件海拔提高到 5500m，并把最高额定电流代码收敛到 3150A。
- WS-G 2024 版标准新增零表压绝缘试验，说明气体绝缘柜还存在低气压/零表压工况下的绝缘验证要求。
- WS-G 40.5kV 图片页把 GIS 价值概括为耐腐蚀、适应恶劣环境、供电连续性高、操作者安全保护高和维护工作量少。
- WS-G 40.5kV 图片页明确其海上风电 35kV 应用语境，并给出双母线 4000A、IAC 40kA/1s 和 [[BLink]] 口径。
- WS-G 2018.09 产品目录则给出更早的常规目录口径：12~40.5kV、最高 2500A、最高 31.5kA 开断、靠墙安装、柜前维护、标准 DXN8B VDS、可选 IVIS，以及 1000m 安装高度。
- WS-G 2024.01 产品目录从产品族角度补充了 12~40.5kV、最高 3150A、最高 40kA 开断、单母线/双母线系统和柜前安装维护口径。
- WS-G 目录强调带电中压部件位于充气密封气室中，不受大气侵蚀、污垢、灰尘和小动物影响，并把 [[BLink]] 作为避免现场气体处理的扩展安装关键。
- WS-G 操作说明书确认气箱按 IEC 62271-200 设计成密封压力系统，绝缘气体为符合 IEC 60376 的 SF6，在正常寿命期和 IEC 60694 正常操作条件下不需要补充绝缘气体。
- WS-G 操作说明书还把 [[IDIS]]、[[IVIS]]/VDS 和 [[机械联锁]] 纳入运行操作边界。
- WS-G 2014 产品介绍从演示材料角度补充了 12~40.5kV、特殊型海拔 4000m、密封气室全寿命免维护、现场安装/扩展/拆卸无需充放气、气体泄漏率 <0.1% p.a. 和自封阀回收再利用口径。
- 2014 材料还用 40.5kV JYN1-35 空气绝缘柜与 WS-G SF6 气体绝缘柜尺寸对比突出 GIS 体积小和运行经济性。
- WSG 特点图片页进一步补充三相共箱式结构、主母线侧/电缆侧电压互感器与避雷器布置、无源气体密度表、750mm 电缆安装高度和 CT 更换无需拆缆等偏工程实施的卖点。
- 同一图片页把气体年泄漏率写为 <0.01%/年，较现有标准与目录常见的 <0.1%/年 更严格，应保留为来源差异而非统一参数。
- WI-G 操作说明书把设备定位为铁道用气体绝缘金属封闭开关设备，额定电压 31.5kV、母线/出线电流 ≤2500A、短路开断电流 ≤31.5kA。
- WI-G 同样采用 SF6 密封压力系统、真空灭弧室、[[BLink]]、[[IDIS]] 和 [[IVIS]]，更强调运行操作、联锁和维护计划。
- `FBX catalogue 2020` 补充了更偏二次配电的气体绝缘路线：[[FBX]] 面向 `24kV` 及以下、`630A`、`20kA` 级应用，分为 `FBX-C` 紧凑型和 `FBX-E` 可扩展型。
- FBX 同样采用 `SF6` 密封压力系统，`20℃` 额定气压约 `0.03MPa`，主回路设计寿命 `30` 年，寿命期内无需补气。
- FBX 目录给出 `LSC2A`、主回路 `IP67`、熔断器隔室 `IP65` 可选 `IP67`，并记录 `AFL/AF 16kA 或 20kA / 1s` 的内部电弧口径。
- FBX 还把 `Easergy T300`、`SMD`、`TH110-R`、`PD110` 和 `VPIS` 纳入可选数字化扩展，说明气体绝缘柜路线也可下探到配电末端的在线监测与远程接入。
- `2018.04` 的另一份 FBX 中文目录说明，这条路线在早期版本中更突出 [[ALink]] 扩展、远方控制等级和联锁逻辑，而尚未像 `2020.05` 版那样系统展开 [[EcoStruxure]] 章节。
- `RM6` 演示稿补入施耐德另一条更成熟的环网柜型二次配电 GIS 路线：覆盖 `12~24kV`、`630A` 主功能单元和 `20kA` 级短路能力，强调全绝缘、可触摸、不锈钢 `SF6` 气箱、年泄漏率 `<0.01%`、`IP4X` 和 `20kA 1s` 内部燃弧口径。
- 与 [[FBX]] 相比，RM6 这份 `2018` 资料更突出环网节点、双电源备自投、`T200C`、`ATS`、`Smart RMU`、`Easergy T300` 和轨道交通共箱方案，而不是以 `FBX-C/FBX-E` 那种标准目录化产品族结构来组织叙事。
- `RM6 SeT` 目录则把这条 RMU / GIS 路线推进到更正式的工程目录阶段：`24kV` 及以下、气箱 `IP67`、电缆室 `IP43`、`IK07`、主回路 `40` 年寿命、`1~6` 功能共箱、`A-link` 无气体处理扩展，以及 `PD110/TH110-R/CL110/SMD/Easergy T300` 数字化配置。
- `RM6-S` 操作说明书则补充这条路线更早的 `12kV` 运维边界：文档继续采用 `SF6` 密封压力系统，给出 `20℃` 时相对充气压力 `0.03MPa`、压力表绿色可操作区、海拔高于 `1000m` 的专用压力表以及“寿命期内通常无需补气”的维护口径。
- 这说明施耐德在二次配电 GIS 领域至少存在两种互相关联但叙事重心不同的目录对象：[[FBX]] 更像标准化产品族目录，[[RM6]] / [[RM6SeT]] 更像围绕环网柜、扩展、自动投切和 Smart RMU 组织起来的平台型路线。

- `RM AirSeT` `2020.07` 资料说明施耐德在 `12kV` RMU / GIS 路线中已有环保气体版本：[[RMAirSeT]] 采用环保气体绝缘、真空灭弧、`GWP < 1`、`IP67` 气箱、`20kA/4s`、`A-FLR 20kA 1s`，并继续保留 `1~6` 功能单元自由组合、`NE/LE/RE/DE` 扩展和 `Easergy T300` 等二次配电叙事。
- 这也让施耐德当前 GIS 路线可以更清楚地分成两组环保分支：[[GMAirSeT]] 偏 `12kV` 一次配电柜型平台，[[RMAirSeT]] 偏 `12kV` 二次配电环网柜 / RMU 平台。
