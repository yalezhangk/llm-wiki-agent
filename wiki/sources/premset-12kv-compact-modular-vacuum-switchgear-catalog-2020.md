## Summary
这份 `SCDOC1783 / 2020.06` 中文产品目录是当前 [[Premset]] 资料链中更新的一版正式样本。它延续 `2018.10` 目录对 [[屏蔽式固体绝缘]]、`12kV`、`630/1250A` 和 `25kA x 1s A-FLR` 的工程口径，同时明显加强了数字化配电自动化叙事，补入 `Easergy T300`、`SMD`、`Easergy P3/P5`、`Sepam`、`VIP 400/410`、`TH110-R`、`Flair 22D/23DM` 等当前施耐德命名体系下的在线监测与配网自动化配置。

## Key Claims
- 封面将 Premset 定义为“`12kV` 紧凑型模块化真空开关设备 / 屏蔽式全绝缘系统 / 产品目录”，末页文档号为 `SCDOC1783`，日期为 `2020.06`。
- 目录延续 Premset 的核心参数口径：`12kV`、`50/60Hz`、`630A / 1250A`、`20kA x 4s`、`25kA x 3s`、`25kA x 4s`、`31.5kA x 4s`，以及断路器 `E2` 级 `25kA 50次` 或 `31.5kA 30次`。
- 文档继续给出 `25kA x 1s A-FLR` 的内部电弧口径，并明确顶部或底部泄压两种实现方式、靠后墙至少 `100mm` 的安装边界，以及底部泄压所需的电缆沟条件。
- 目录把 Premset 的智能化方案扩展成完整架构：通信网络可采用无线广播、`2G/4G`、光纤以太网，系统覆盖故障指示、智能电表、保护继电器、开关监控、综合管理单元和后备电源管理。
- `Easergy T300` 被写成分布式模块化智能配电终端，支持 `IEC 60870-101/104`、`MODBUS` 和 `IEC 61850`，面向馈线自动化、负荷管理、资产管理和双电源/发电机/母联备投。
- 智能设备清单补入 `SMD`、`Easergy P3/P5`、`Sepam`、`VIP 400/410`、`Flair 22D/23DM`、`SC110/150/160`、`HU250`、`PS100/PS50`、`TH110/TH110-R/CL110`、`VPIS`、`VD23`、`EcoStruxure Power Device` 和 `eSetup Easergy Pro`。
- 目录比 `2018.10` 更清楚地解释了柜型编码：`I` 为负荷开关柜，`D` 为断路器柜，`G` 为直连或提升柜，`M` 为计量柜；`06` 代表 `630A`，`12` 代表 `1250A`，`T` 代表 `CIT` 机构，`H` 代表 `OCO` 机构；`M` 柜末位 `A` 代表空气绝缘，`S` 代表固体绝缘。
- 结构页补充了嵌入式电流互感器、模块化母线系统、集成空气绝缘接地开关、前插电缆连接、内燃弧泄压通道、可选母线电压互感器 `LVPT`，以及无线无源温度传感器 `TH110-R`。
- 环境与安装边界比 `2018.10` 更完整：总高度 `1550mm~2195mm`，`630A` 固体绝缘单元标准宽度 `375mm`，部分 `1250A` 与空气绝缘计量/进线计量单元为 `750mm`；系统深度 `910mm`、`1135mm` 或 `1251mm` 取决于是否带内燃弧通道和后进线方案。
- 目录明确给出外壳防护等级 `IP4X`，并注明 `IP32~IP42` 可按配置定制；还补充 `IEC 62271-210 0.33g`、`ASCE 7-10 / IBC 2012 / CBC 2013 / ICC-ES AC156 等级3` 抗震口径，以及 `IEC 60068-2-52` 严酷等级 5、4 周期、28 天盐雾口径。
- 文档延续并正式写出“浸水后可连续服务 `48小时`（仅限固体绝缘柜型）”“`5000m` 以下无需额外预防措施”“`5000m` 以上每升高 `1000m` 绝缘降容 `10%`”等特殊环境应用边界。
- 目录补充了海事 `DNV-GL` 应用认证，以及商业楼宇、工业、配电网络、中低压环网、中压分布式电源、水务、管廊、电网核心供电区域等更细应用场景。

## Contradictions
- 与 `2018.10` 的 [[Premset]] 目录没有直接冲突；`2020.06` 版延续了核心电气参数和 [[屏蔽式固体绝缘]] 路线，但明显强化了数字化与应用场景叙事，可视为更新版本而不是替代完全不同产品。
- 与 `2012.06` 的 `Premset 产品手册` 相比，`2020.06` 版不再只是“智能电网就绪”的高层宣传，而是落到 `Easergy T300`、`SMD`、`TH110-R`、`Flair 22D/23DM` 等具体设备名和通信协议。
