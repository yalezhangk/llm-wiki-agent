## Summary
这份英文 PDF 适合按 `2017-08-21` 的 DNF7 选型手册层处理，而不是当作已入库早期英文技术说明或 `2025-04-10` 中文样本的重复件。它的技术主体与较早的 `DNF7 Metal-Clad Switchgear Technical Instruction` 高度同源，继续围绕 `36kV / 40.5kV`、四隔室、`LSC2B / PM`、`HVX / FP` 和两种抽出式方案展开。它真正新增的价值，是把这套较早 DNF7 技术骨架明确带到 `2016/2017` 施耐德品牌与公司介绍口径下，形成中间版本层。

## Key Claims
- 文档封面仍把 DNF7 定义为 `Metal-Clad Switchgear / Air Insulated Switchgear`，说明路线未改名为新平台。
- 技术主体延续较早英文版：`36kV / 40.5kV`、四隔室、`LSC2B`、`PM partition class`、`HVX / FP`、`Middle Rolling / Floor Rolling`。
- 工程口径继续保留 `AFLR 31.5kA/1s`、`IP4X` 可选 `IP41`、`1200(1400) x 2505 x 2400 mm` 和最大 `1800kg`。
- 公司介绍页升级到 `2016` 财年与 `2017` 时点的话术，包括 `24.7 billion Euros in 2016`、`160,000 plus employees`、中国 `26,000` 员工和“over 30 years”在华表述。
- 文末仍保留 `ONEXXXX 2010.12` 尾标，说明这更像对较早 DNF7 技术骨架的英文再包装，而不是全新技术版。
