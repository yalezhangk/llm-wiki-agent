## Summary
这份 `SCDOC1544 / 2020.05` 目录介绍了 [[FBX]] 中压 [[气体绝缘开关柜]]，定位为 `24kV` 及以下二次配电用紧凑型设备。它补充了与 [[GMA]]、[[GHA]] 和 [[WSG]] 不同的一条施耐德产品线：FBX 更强调 `630A / 20kA`、紧凑柜宽、变压器保护单元、`LSC2A` 连续性和寿命期内免补气，同时保留可接入 [[EcoStruxure]] 的在线监测选项；与更早的 `2018.04` 版相比，这一版更明显地强化了数字化章节。

## Key Claims
- [[FBX]] 面向公共配电、工业、基础设施和可再生能源场景，覆盖 `12kV` 和 `24kV`，额定电流 `630A`，额定短时耐受/内部电弧主口径为 `20kA` 级。
- FBX 采用 `SF6` 绝缘，带电主回路封闭在不锈钢气箱中，主回路设计寿命 `30` 年，寿命期内免维护、无需补气。
- 产品分为 `FBX-C` 紧凑型和 `FBX-E` 可扩展型；前者可把 6 个以下功能单元集成在单个气箱中，后者支持左右扩展。
- `FBX-E` 通过 [[ALink]] 实现左右扩展，扩展时无需现场气体处理。
- 典型功能单元包括 `C` 进出线、`T1` 熔断器变压器保护、`T2` 真空断路器线路/变压器保护、`R` 直接进线、`Sb` 母线分段和 `M` 计量单元。
- 目录给出 `LSC2A` 工作连续性、主回路 `IP67`、熔断器隔室 `IP65` 可选 `IP67`、操作机构和低压隔室 `IP3X`、电缆隔室开门 `IP2XC` 和柜体 `IK07`。
- `T1` 单元强调上下游双接地开关、熔断器撞针触发三相分闸和机械联动；`T2` 单元强调固定式真空断路器、具有关合能力的接地开关和可选无辅助电源故障跳闸继电器。
- 目录给出 `AFL/AF 16kA 或 20kA / 1s` 的内部电弧口径，并提供室内安装、泄压和运输通道要求。
- EcoStruxure 章节补充 `Easergy T300`、`SMD`、`TH110-R`、`PD110` 和 `VPIS` 等数字化监测与接入选项。
- 可持续发展章节说明约 `90%` 金属材料可回收，`SF6` 约占总重 `0.5%`，可在寿命期终通过阀门回收再利用。

## Contradictions
- 未发现与 [[GMA]]、[[GHA]] 或 [[WSG]] 的直接冲突；它们同属施耐德中压开关柜，但 FBX 当前来源更偏 `24kV` 及以下、`630A / 20kA` 的二次配电紧凑路线，不应与更高电流或更高电压产品线参数互相覆盖。
- 与 [[FBX]] 的 `2018.04` 目录相比，这一版延续同一 `SCDOC1544` 口径，但新增了更明显的 [[EcoStruxure]]、`Easergy T300`、`SMD`、`TH110-R` 和 `PD110` 数字化内容。
