## Summary
这份 `68` 页中文 PDF 是 [[RM6]] 路线一层独立的正式产品目录，而不是现有 `2018-05-09` 中文演示稿的重复件。它带有 `SCDOC1043-MV / 2018.10` 尾标，结构完整覆盖设备特性、附件、电网远程控制、中压电缆头、户外开关站、安装、订购表和方案页，因此更适合作为 RM6 在中国市场的工程目录层来理解。

## Key Claims
- 封面写有 `RM6 中压开关柜`、`SF6绝缘环网柜和紧凑型开关柜`、`3～24kV`、`产品目录`，PDF 元数据显示创建和修改时间均为 `2018-09-28`。
- 末页尾标为 `SCDOC1043-MV / 2018.10`，说明它是正式目录层，而不是培训稿、演示稿或操作手册。
- 公司页保留 `2017` 财年施耐德口径：`247 亿欧元`、全球 `14.2 万` 员工、中国 `17000+` 员工、`23` 家工厂、`8` 个物流中心、`9` 个分公司和 `37` 个办事处。
- 产品定义页把 RM6 写成满足 `24kV` 以下中压配电系统需求的设备，可保护开放式环网或辐射式网络中的变压器，且带电部件永久封闭在 `SF6` 壳体内。
- 路线页强调 RM6 全球已有 `170 万台` 功能单元应用，并写出 `2016：诞生了智能化环网柜` 这一历史节点。
- 文档给出 `1~5` 功能模块、`2/3/4/5` 回路连接、`630A` 断路器线路保护和可集成远程控制设备的总体产品框架。
- 环境与结构参数页明确写出 `-40℃ ~ 40℃`、海拔 `≤1000m`、`SF6气室 IP67`、熔断器室 `IP67`、外壳 `IP4X`、`20℃下 SF6充气压力 0.2Bar`、年泄漏率 `0.01%/年` 和气室不锈钢板厚 `2mm`。
- 电气特性表覆盖 `12kV / 24kV` 两档，给出工频耐压 `42/48kV` 与 `65/79kV`、冲击耐压 `95/110kV` 与 `125/145kV`、主回路额定电流 `630A`，并把 `20kA/4s`、`20kA/3s`、`25kA/1s` 等选项落到订购层。
- 功能体系页明确列出 `I / B / D / Q / IC / BC / O / Pt / Mt`，柜型体系明确列出 `NE / RE / LE / DE`。
- 主要特性页把内部燃弧能力写为 `20kA 1s`，并把气压表/压力开关、门连锁、低压间、升高座和 `Fusarc CF` 列为正式选件。
- `I` 功能附件页列出电动操作、辅助触点、`Flair 22D`、`Amp 21D` 和钥匙锁；`B` 功能页列出 `VIP400 / VIP410 / Sepam 20系列`。
- 自动投切页把双电源自动切换系统和网络 `ATS` 正式写入目录，控制器明确为 `T200 I`，并给出 `TR < 2 s`、`T1 0s~200s`、`T2 0s~30分钟` 以及自动/半自动模式。
- 附件页继续补入 `Flair 22D / Flair 23DV / Amp 21D`、`VD23`、`VPIS`、压力表/压力开关，以及电缆室内部电弧 `20kA 1s` 和钥匙锁/机械联锁逻辑。
- 订购表进一步确认这是一层正式工程目录，明确给出 `12kV 20kA/4S`、`12kV 20kA/3S`、`12kV 25kA/1S`、`24kV 20kA/1S`、`24kV 20kA/3S`、`Flair 22D`、`Amp21D`、`VPIS + RV10`、`Sepam`、`VIP45` 和 `VIP400` 等订购项。
