## Summary
这份 `2020.07` 中文资料把 [[RMAirSeT]] 定位为施耐德 RM 路线中的环保气体 [[气体绝缘开关柜]] / 环网柜分支。它不是 [[GMAirSeT]] 那种一次配电平台，也不能简单视作 [[RM6SeT]] 的改名版本；更稳妥的理解是施耐德在 `12kV` 二次配电 RMU 场景下并行布局的一条 `GWP < 1` 环保气体绝缘、真空灭弧路线。资料同时把 `Easergy T300`、`PMBOX`、`PD110`、`TH110-R`、`VT/LPVT`、`CT/LVCT` 等数字化配置直接写入产品叙事，说明环保气体路线也已经与主动运维和配网自动化结合。

## Key Claims
- [[RMAirSeT]] 面向 `12kV` 中压二次配电场景，采用环保气体绝缘和真空灭弧，封面强调 `GWP < 1`、全生命周期碳足迹降低和不燃。
- 核心开断与操作结构包括三工位负荷开关、负荷开关-熔断器组合电器、`V/B` 真空断路器、接地开关和母线隔离，且提到 `SVI` 并联真空开断技术。
- 功能单元体系明确写为 `C / F / V / B`，并补充 `PT`、计量柜和母联等扩展对象。
- 柜列支持 `1~6` 功能单元自由组合，扩展代号为 `NE / LE / RE / DE`，显示它延续的是 RMU 平台化组合逻辑。
- 技术参数页给出 `12kV`、`50Hz`、工频耐受 `42/48kV`、雷电冲击 `75/85kV`、额定短路开断电流 `20kA`、额定短时耐受 `20kA / 4s`、峰值耐受 `50kA`。
- 内部电弧能力写为 `A-FLR 20kA 1s`，防护等级写为 `IP4X / IP67`。
- 气体年泄漏率给为 `≤0.025%`，额定充气压力给为 `0.4 bar`，并配置气压表。
- 额定电流与寿命口径包括：`C` 单元 `630A`、`F` 单元 `125A`、`V/B` 单元 `630A`；`C/F` 机械寿命 `5000` 次，`V/B` 机械寿命 `10000` 次，`V/B` 短路开断次数 `30` 次。
- 文档给出 `O - 0.3s - CO - 180s - CO` 自动重合闸顺序，以及 `F` 单元 `1750A` 转移电流。
- 典型尺寸显示单元柜约 `420mm x 850mm (780mm 可选) x 2000mm`，带 `350mm` 低压室；部分一二次融合/PT 柜为 `600mm x 850mm (780mm 可选) x 2000mm`。
- 使用环境口径为 `-25℃ ~ +40℃`、湿度 `95% / 24h`、`90% / 1 month`、常规海拔 `1000m`，更高海拔需咨询施耐德。
- 数字化章节直接列出 `Easergy T300`、`Modbus`、`HMI`、`PMBOX`、`PD110`、`TH110-R`、`VT/LPVT`、`CT/LVCT`，并围绕实时感知、状态监测、资产预警和预测性资产管理组织卖点。

## Key Quotes

> "最多6功能单元，自由组合" — 柜列组合能力概述。

## Contradictions
- 与 [[GMAirSeT]] 不构成直接冲突；两者更适合视为施耐德环保气体路线在一次配电与二次配电环网柜上的不同平台。
- 与 [[RM6SeT]] 也不应处理为简单新旧替代关系：[[RMAirSeT]] 这份 `2020.07` 资料强调 `12kV`、`C/F/V/B`、`NE/LE/RE/DE` 和环保气体 `GWP < 1`，更像 RM 路线中的平行技术分支。
