## Summary
这份 `SCDOC1541 / 2018.09` 产品目录是当前 wiki 中早于 `2024.01` 版的 [[WSG]] 产品级资料。它从概述、优点和改善、关注细节、运行条件与技术特性、尺寸重量、主接线方案、应用范例以及发货运输订货要求等角度，给出一个偏工程选型的早期口径：额定电压至 `40.5kV`、额定电流至 `2500A`、额定开断电流至 `31.5kA`，并强调 [[BLink]]、[[IDIS]]、`DXN8B` VDS、可选 [[IVIS]] 和靠墙安装等特征。

## Key Claims
- 封面标题为“交流金属封闭气体绝缘开关柜 WS-G 12~40.5kV 产品目录”，末页印有文档号 `SCDOC1541` 和日期 `2018.09`。
- 目录将 WS-G 定位为“安全、可靠、环保”以及“简单、创新、经济”的气体绝缘开关柜，适用于公共和工业配电网、基础设施工程、采矿、冶金、石油化工、燃油燃气、铁路供电、集装箱基地和船舶行业。
- 这版目录的产品口径是额定电压可至 `40.5kV`、额定电流可至 `2500A`、额定开断电流可至 `31.5kA`，可设计为单母线或双母线系统。
- 用户受益部分强调：扩展和柜子更新时无需气体处理、创新的耐故障母线连接、友好的操作界面、操作可靠性和人员操作安全、寿命期成本低以及环保易回收。
- [[BLink]] 被描述为创新母线连接系统：每台柜母线位于独立充气室内，现场安装容易且无需气体处理；扩展或更换开关柜时不影响气室；硅橡胶绝缘件在工厂已安装并做局放试验；现场安装可视；拆卸后可形成绝缘距离；必要时可分段测量母线电阻。
- 目录强调 WS-G 可标准靠墙安装，不需要安装通道，所有操作和维护过程都可在柜前完成，也可选择独立式安装方式。
- 通用模块包括：内锥/外锥进出线断路器柜、高压出线侧带隔离/接地开关的 PT、双母线母线联络合柜、带断路器的母线分段和提升合柜、母线分断柜、计量柜，以及 PT、接地开关、避雷器和母线带电显示等母线辅助模块。
- 断路器柜说明中，真空断路器安装在金属封闭充气室内；单母线模块装三工位隔离/接地开关和母线系统；双母线模块由上部气室三工位开关和后部气室二工位隔离开关构成。
- 目录把问答式机械联锁写为关键安全特征：断路器弹簧机构支持连续自动重合闸，断路器与二/三工位隔离接地开关的问答式机械联锁可避免误操作。
- [[IDIS]] 被明确写为每个气室的气体密度监控系统：气体状态由压力传感器检测，经电子信号送到柜前 IDIS 显示器；每个气室监测信号独立；一个显示器最多可连接三个压力传感器；报警信号可通过接点远传。
- 目录给出额定充气压力、预警压力和报警压力：`12~24kV` 为 `0.030 / ≤0.030且>0.010 / ≤0.010MPa`，`36kV` 为 `0.050 / ≤0.030且>0.010 / ≤0.010MPa`，`40.5kV` 为 `0.060 / ≤0.050且>0.030 / ≤0.030MPa`。
- 电压检测采用 VDS 方案：带电测试通过电缆插座内的电容分压器实现，标准配置为 `DXN8B` 户内高压带电显示装置；也可选用 [[IVIS]] 智能化电压检测系统。
- 电缆连接系统给出外锥和内锥两条路线：外锥按 `EN 50181 C` 型、适用于 `33kV` 电网、最高只能达到 `36kV`；内锥按 `EN 50181`、适用于 `35kV` 电网，并可配 2/3/4 号终端与避雷器。
- 运行环境条件按 IEC 62271-1 / IEC 60694 口径给出：最高温度 `40℃`、24 小时平均 `35℃`、室内最低 `-5℃/-25℃`，安装高度 `1000m`，更高值按要求提供。
- 技术特性表给出：雷电冲击耐受 `75/125/170/185kV`，工频耐受 `28/42, 50/65, 70, 95kV`，峰值耐受 `100kA`，短时耐受电流最大 `40kA`（40.5kV 为 `40kA/4s`），额定母线电流和出线柜额定电流最大均为 `2500A`，抗内部电弧等级最大 `40kA 1s`。
- 尺寸表给出单母线与双母线在 `12~36kV` 和 `40.5kV` 下、`800~1600A / 2000A / 2500A` 不同电流等级对应的柜宽、柜深和柜高；重量表给出外锥出线柜约 `700kg`、内锥出线柜约 `1000kg`、内锥进线柜约 `900kg`、母线分段柜约 `1100kg`。
- 目录列出典型单母线和双母线主接线方案，并补充 PT、接地开关、避雷器、母线带电显示等母线辅助模块，以及土建基础图和空间分布图。
- 发货文件包括合格证、出厂检验报告、装箱与备件清单、使用说明书、设备规范表和辅助回路接线图；订货资料要求电气参数、单线图、保护控制监测资料、布置图、门廊和运输通道尺寸等。

## Contradictions
- 这份 2018 目录把 WS-G 常规产品口径写为 `2500A / 31.5kA / 1000m`，而 `2024.01` 产品目录与 `2024` 企业标准已更新为最高 `3150A / 40kA / 5500m` 的口径；它们应视为版本演进而非简单冲突。
- 2015 企业标准和部分图片页保留 `4000A` 主母线或双母线应用口径，而这份 2018 目录与 2024.01 目录都未把 `4000A` 作为常规参数表主口径；工程引用时需要区分“标准代码/特定方案”和“目录常规产品”。
- 本目录将 `IVIS` 作为可选项、标准带电显示为 `DXN8B` VDS；操作说明书则把 `IVIS/VDS` 放在统一电压检测系统章节下描述，属于资料角度差异，不构成功能冲突。
