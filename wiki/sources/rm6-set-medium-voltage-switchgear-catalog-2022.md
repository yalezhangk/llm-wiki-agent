## Summary
这份 `ECATA1138 / 2022.02` 中文产品目录把 [[RM6SeT]] 定义为施耐德 `24kV` 及以下中压二次配电 [[气体绝缘开关柜]] / 环网柜方案。与已入库的 `2018` 年 [[RM6]] 演示稿相比，它更像正式工程目录：补全了 `IP67` 气箱、`IP43` 电缆室、`IK07`、`40` 年寿命、`A-link` 扩展、`1~6` 功能共箱、`C/F/V/D/PT` 功能体系、`Easergy P1` 保护和 `PD110/TH110-R/CL110/SMD/Easergy T300` 数字化方案。

## Key Claims
- 封面把产品定义为 `SeT` 系列 `RM6 SeT` 中压开关柜，副标题为“`24kV` 及以下气体绝缘开关设备”，末页文档号为 `ECATA1138 2022.02D0214`。
- 目录强调主回路 `40` 年以上使用寿命、寿命期内免维护、开关柜在寿命期内无需补气，`3mm 304` 不锈钢气箱保证极低气体泄漏率。
- 文档将 RM6 SeT 描述为“鲁棒性设计、全能型架构、数字化”，强调适用于恶劣环境、紧凑型设计、三工位负荷开关、双电源自动切换、闭环配电网和自动重合闸。
- 资料称施耐德已有超过 `180` 万台中压气体绝缘开关柜在 `150` 多个国家和地区运行，并把 RM6、FBX、GMA、GHA、MCSet/F400 和 `Easergy T300` 放入同一历史演进图谱。
- RM6 SeT 可实现 `1~6` 功能共箱，并支持不可扩展 `NE`、左扩 `LE`、右扩 `RE` 和双扩 `DE`；扩展通过 [[ALink]] 实现，扩展时无须气体处理、无须特殊工具、无须特殊地基准备。
- 对自由组合的限制写得更明确：`PT` 电压互感器柜和 `M` 计量柜只能单元式、不能共箱，只能通过可扩展 RM6 SeT 连接。
- 主要一次功能体系包括 `C` 三工位负荷开关、`F` 负荷开关-熔断器组合电器、`V` 真空断路器、`D` 提升柜和 `PT` 辅助电源/母线电压测量柜。
- 基础绝缘参数为 `12kV / 24kV`，相间及对地工频耐受 `42 / 65kV`，断口工频 `48 / 79kV`，雷电冲击 `75/95 / 125kV`，断口冲击 `85/110 / 145kV`。
- 母线额定电流为 `630A`，短时耐受电流 `20kA`，持续时间 `4/2s`，额定峰值耐受电流 `50kA`。
- 外壳与隔室防护边界比旧 `RM6` 演示稿更完整：气箱 `IP67`，电缆室 `IP43`，机构室 `IP4X`，低压间 `IP4X`，隔室间 `IP2X`，机械冲击等级 `IK07`。
- 内部电弧一栏写明基础型为“未分类”，同时提供 `A FL` 与 `A FLR 20kA 1s` 口径，说明 `IAC A-FLR` 是可选项而不是所有基础柜型默认分类。
- `C` 功能负荷开关页给出 `M2` 级 `5000` 次机械寿命、`E1/E2/E3` 开断等级、`C1/C2` 容性电流开断重击穿等级，以及接地开关 `3000` 次延长机械寿命。
- `F` 功能熔断器组合电器页给出 `31.5kA` 开断能力，转移电流 `12kV 1750A`、`24kV 1000A`，并注明可选 `1200A`。
- `V` 功能真空断路器页给出 `20kA` 开断能力、`M2 10000` 次机械寿命、`E2` 电寿命、分闸/跳闸时间 `<40ms`、合闸时间 `<60ms`，以及 `O - 0.3s - CO - 3 min - CO` 自动重合闸顺序。
- 保护页把 `VIP40/45/400/410` 与 `Easergy P1` 并列，说明 RM6 SeT 已不只依赖早期 VIP 路线，还可接入更完整的 `Easergy P1` 保护 IED。
- 数字化页补入 `Easergy T300`、`VT/LPVT`、`CT/LVCT`、`TH110-R`、`PD110`、`CL110`、`PS100/PS50` 和 `SMD`，并接入 [[EcoStruxure]]、`Asset Advisor`、`ADMS`、`Substation Operation` 等架构。
- 目录明确 `Easergy T300` 支持 `IEC 60870-101/104`、`MODBUS`、`IEC61850` 和 `IEC61131-3` 自动化逻辑，兼容电脑、手机和移动终端运维。
- 安装页给出更正式的尺寸口径：单功能 `C/D` 柜宽可到 `375mm`，`V/F/PT420` 常见宽度 `420mm`，`PT600` 宽 `600mm`；最大 `6` 功能共箱，总宽可到 `2170mm`，深度 `750mm`。
- 土建页给出到后墙 `≥20mm`、不可扩展侧面到墙 `≥30mm`、可扩展侧面到墙 `≥450mm`、到屋顶 `≥100mm` 等安装边界，并补充防内燃弧站房的天花板高度和泄压条件。

## Contradictions
- 与 `2018` 年 [[RM6]] 演示稿没有直接冲突；`RM6 SeT` 目录可以看作 RM6 路线更正式、更工程化的一版资料。两者共通点是 `12~24kV`、`630A`、`20kA` 级二次配电 GIS / RMU 口径，但 `RM6 SeT` 更强调 `1~6` 功能共箱、`A-link` 扩展、`IP67/IP43/IK07`、`Easergy P1` 和完整 [[EcoStruxure]] 数字化方案。
- 与现有 [[FBX]] 相比，二者都属于施耐德中压二次配电 GIS，但 RM6 SeT 更偏环网柜与自由组合/扩展平台，FBX 更偏 `FBX-C / FBX-E` 产品族和 `LSC2A` 目录表达。
