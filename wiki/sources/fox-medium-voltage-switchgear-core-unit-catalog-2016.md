## Summary
这份 `SCDOC1608 / 2016.03` 中文 PDF 不是完整环网柜目录，而是 [[FOX]] 这条“中压开关柜核心元件”路线的正式产品目录。资料把它明确写成 `12kV` 以下、`SF6` 气体绝缘的开关柜核心单元，适用于配电网络环网单元和最终用户配电，并强调由施耐德提供核芯元件、由合作伙伴完成电缆室和低压间。与现有 [[AS12AS24]]、[[SC6]] 这种“空气绝缘柜体中的核心开关元件”路线相比，[[FOX]] 更像一条面向二次配电 GIS / RMU 集成场景的成套核心模块路线。

## Key Claims
- 资料封面写为 `FOX 中压开关柜核心元件 12kV 气体绝缘开关设备`，末页尾标为 `SCDOC1608 / 2016.03`。
- [[FOX]] 被定位为 `12kV`、`630A` 的中压开关柜核心元件，可作为配电网络环网单元或最终用户配电单元。
- 产品有两种柜型：`FOX-NE` 紧凑型和 `FOX-DE` 可扩展/模块化柜型；`FOX-DE` 支持左右两侧扩展。
- 保护方案分为 `Q` 方案和 `B` 方案：前者用负荷开关-熔断器组合电器做变压器保护，后者用真空断路器做变压器保护。
- 资料给出 `I / Q / B / O / IC` 功能单元体系，其中 `B` 单元适用于负荷电流较大、不能使用负荷开关-熔断器组合的场合。
- 技术参数页给出：`12kV`、工频耐压 `42kV`、雷电冲击 `75kV`、母线 `630A`、额定短时耐受 `20kA / 4s`、峰值耐受 `50kA`、`B` 型柜短路开断 `20kA`、`20℃` 时 `SF6 0.03MPa`。
- 目录强调 `30` 年使用寿命，寿命期内主回路免维护，并通过询问式机械联锁系统避免误操作。
- 外形图页明确保留制造边界：核芯元件由施耐德提供，电缆室和低压间由合作伙伴生产。

## Key Quotes
> "FOX为中压开关柜核心元件，12kV，630A。"

> "以上核芯元件由施耐德电气提供，电缆室和低压间由合作伙伴生产。"

## Connections
- [[FOX]] — 这份资料是其独立正式目录层。
- [[施耐德电气]] — 核芯元件提供方，资料使用 `2014` 财年中国市场公司介绍口径。
- [[气体绝缘开关柜]] — 作为 `12kV` 二次配电 GIS / RMU 核芯模块路线的具体实例。
- [[机械联锁]] — 资料强调询问式机械联锁系统和清晰可见的状态模拟图。
- [[AS12AS24]] — 同样属于“施耐德提供核心元件、合作伙伴完成柜体”的路线，但后者偏空气绝缘柜体集成。
- [[SC6]] — 可与 `AS12AS24 -> SC6` 的收敛关系对照理解，都是柜厂集成型元件路线。

## Contradictions
- 与现有 [[FBX]]、[[RM6]]、[[FLUSARC]] 不构成直接冲突；这些页面更偏完整产品线或整柜目录，而这份资料强调“核芯元件 + 合作伙伴柜体”的边界。
- 与 [[AS12AS24]]、[[SC6]] 的差异不在于是否为“核心元件”，而在于绝缘路线不同：[[FOX]] 是 `12kV` 以下 `SF6` 气体绝缘开关设备，后两者更偏空气绝缘柜体中的中压开关核心。
- `SCDOC1608_web - FOX.pdf` 与当前来源在页数、关键正文、目录号 `SCDOC1608` 和时间 `2016.03` 上一致，但文件哈希不同，更适合视为同版 `web` 导出副本而非新的版本层。
