## Summary
这份 `SCDOC1544 / 2018.04` 中文目录是当前 [[FBX]] 资料链中更早的一版正式产品目录。它与后续 `2020.05` 版共享 `24kV` 及以下、`630A / 20kA`、`FBX-C / FBX-E`、`LSC2A` 和 `AFL/AF 16/20kA 1s` 等核心工程口径，但更突出 [[ALink]] 扩展结构、远方控制等级、联锁逻辑和安装实现，而尚未出现后续版本更明显的 [[EcoStruxure]] 监测与接入章节。

## Key Claims
- [[FBX]] 被定义为 `24kV` 及以下、`630A`、`20kA` 的二次配电用中压开关装置，可通过 `T1` 熔断器或 `T2` 真空断路器实现变压器保护。
- 目录给出 `FBX-C` 紧凑型和 `FBX-E` 可扩展型两条柜型路线，并列出 `C`、`T1`、`T2`、`R`、`Sb` 和 `M` 功能单元。
- [[ALink]] 被明确写为 `FBX-E` 的特有母线连接装置，支持左右扩展、无需现场气体处理，并要求扩展时预留大于 `450mm` 的安装空间。
- 目录强调主回路 `30` 年使用寿命、寿命期内免维护、无需补气，以及 `LSC2A` 类工作连续性。
- 运行条件包括常规海拔 `1000m`、`SF6` 绝缘、`20℃` 额定压力 `0.03MPa`、主回路 `IP67`、熔断器隔室 `IP65` 可选 `IP67`、低压隔室 `IP3X` 和柜体 `IK07`。
- 内部故障电弧等级给出 `C/T1/T2/R` 的 `AFL 16kA 或 20kA / 1s` 与 `M1~M4` 的 `AF 16kA 或 20kA / 1s`。
- 使用章节系统列出隔离开关、接地开关、断路器和电缆隔室门的联锁状态，并说明只有在允许状态下才可插入操作手柄。
- 目录提供远方控制 `0~4` 级口径、`VDS-HR`、`VDS-LR`、`VPIS` 和可选 `IVIS`，但尚未像后续版本那样展开 [[EcoStruxure]] 与在线监测设备清单。
- 末页给出 `2018.04` 日期、客户关爱中心热线与“本手册采用生态纸印刷”等早期发布特征。
