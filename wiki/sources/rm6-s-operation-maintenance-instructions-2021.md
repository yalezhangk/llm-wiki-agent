## Summary
这份 `RM6-S` 中文《操作说明书》不是产品目录，而是一份面向运行、操作与维护的 `12kV` [[气体绝缘开关柜]] 说明书。它把 [[RM6S]] 放在更早期的 `RM6` 路线中，重点讲 `OE / I / Q / B / IC` 功能单元的手动/电动操作、[[机械联锁]]、可视接地、`Flair2XD`、`VPIS-V2`、`T200I`、`VD23`、电缆试验和 `SF6` 压力维护边界，而不是像后续 [[RM6SeT]] 目录那样提供完整产品族参数表。

## Key Claims
- 封面标题为 `RM6-S 气体绝缘金属封闭开关设备 / SF6 Gas-Insulated Switchboards 12kV / 操作说明书`，说明文档对象是 `12kV` RM6-S。
- 页脚与末页给出 `SAS0.463.171-00`、`AMTNoT132-02` 和侧边 `SAS0.463.171-00 (2021/2)` 口径，表明这是一份较晚修订仍沿用较早模板的运维文件；正文同时保留 `© 施耐德电气 2010` 版权标识。
- 目录说明文档主体不是参数总览，而是运行维护结构：服务说明、关于说明书的说明、联锁、操作附件、`OE / I / Q / B / IC` 功能单元操作、电动操作、可视接地、`Flair2XD`、`VPIS-V2`、`T200I`、`VD23`、维护、备件、电缆试验、`SF6` 气体特性和寿命终结。
- 文档明确要求操作前细读说明书，并说明这里只描述设备不带电安装态或禁止操作状态下的操作；操作人员必须经过安全培训并佩戴适当防护装置。
- 联锁章节单独区分 `Q`、`B` 和 `IC` 功能单元，说明 RM6-S 的运行安全高度依赖机械联锁而不是仅靠操作经验。
- `OE / I / Q / B / IC` 五类功能单元都有独立操作章节，说明 RM6-S 采用的是比 `2018` RM6 演示稿和 `2022/2025` RM6 SeT 目录更早的一套功能代号体系。
- `B` 功能单元操作页显示断路器合闸/分闸与隔离开关合闸/分闸之间存在明确顺序关系，并支持可选按钮或电动操作分闸。
- 电动操作章节区分可选电动操作机构、带电动控制的应急手动操作，以及不同功能单元的手动接地开关分闸方法。
- 文档设有“可视接地”专章，说明 RM6-S 把接地可见性作为专门操作对象，而不只是接地状态指示。
- `Flair2XD` 章节把 `Flair 21D / 22D / 23DV` 作为故障电流指示模块引入 RM6-S；资料说明 `21D` 可由传感器自供电，而 `22D / 23DV` 可接入 `VPIS-V2` 带电指示输出并需要 `24Vdc~48Vdc` 外部供电。
- `Flair2XD` 的 CT 与中压电缆连接页给出安装边界：电缆长度 `3m` 以下、电缆截面积 `1.5mm²` 以下，并要求 `MF1 CT` 三个 CT 安装方向一致且接地编织电缆穿过 CT 内。
- `VPIS-V2` 章节说明其内置 `3` 个指示灯，符合 `IEC 61958`；但手册明确警告：单独由 `VPIS-V2` 提供的电压指示不能确认系统已经失电。
- 同一章节说明 `VPIS-V2` 还提供与核相器连接的触点，核相器型号给为 `V2-VPI62421`。
- `VD23` 章节把它定义为电压检测继电器，输入来自 `VPIS-V2`，外部供电为 `24V` 或 `48V` 直流；它支持自动校准、过电压/欠电压检测和相序检测，并通过 `R1 / R2` 继电器与时间延迟参数工作。
- `T200I` 章节不是产品宣传，而是界面、检测与控制原理和功能测试手册；页面显示其测试覆盖数字输入、数字输出和模拟输入，说明 RM6-S 已有比较明确的二次控制接入边界。
- 电缆试验章节给出“母线通电时使用插入式三通接头进行电缆试验”的步骤，包括安装测试适配器、下压联锁滑片模拟电缆室门解锁、打开接地开关、执行试验和恢复使用。
- 同一章节特别警告：在这种电缆试验操作条件下，负荷开关不再受联锁保护；电缆室门和负荷开关之间可选增加附加联锁机构。
- `SF6` 章节说明绝缘气体符合 `GB/T 12022`，每台设备包含一个按 `GB 3906` 充气并设计为气密密封压力系统的气箱，正常工作寿命期内不需要再补充充气。
- 手册给出 `20℃` 时相对充气压力 `0.03 MPa`、绝对压力 `0.13 MPa` 的口径，并展示普通温度补偿压力表和海拔超过 `1000m` 的专用压力表。
- 压力表触点动作阈值表给出 `20℃` 下高阈值 `250 ± 30 mbar`、低阈值 `140 + 50 mbar`。
- `RM6-S` 开关柜只有在压力表指针位于与环境温度对应的绿色区域内时，才能操作负荷开关；指针落入红色区域时需联系施耐德代表。
- 文档反复强调安全禁令：不能在有压力的气箱上冲孔、不要试图打开气箱、不要在没有专用工具和专业人员时回收 `SF6`。
- “设备寿命的终结”章节给出样例 `RM6-S-NE I-I-Q (12kV - 630A)` 的退役评估，并在材料再分配表中列出总重 `310.392 kg`、其中金属部分 `275.991 kg / 88.92%`、`SF6 2.450 kg / 0.79%`，显示这份说明书兼顾了寿命期末回收边界。

## Contradictions
- 与 `2018` 年 [[RM6]] 演示稿、`2022/2025` 年 [[RM6SeT]] 目录并不直接冲突，但资料层级完全不同：RM6-S 是 `12kV` 运维说明书，后两者是产品宣传或工程目录，不能拿本手册的窄口径去覆盖后续产品族参数。
- RM6-S 的功能代号体系为 `OE / I / Q / B / IC`，明显不同于 `2018` RM6 演示稿的 `I / IC / B / BC / D / Q / Mt / O` 和 `RM6 SeT` 的 `C / F / V / D / PT / Cr / Vr / M`。这更像版本与平台演进，而不是简单名称缩写差异。
