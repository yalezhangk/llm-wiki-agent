## Summary
这份 `ECATA1138 / 2025.02` 中文目录是 [[RM6SeT]] 的后续修订版，不是已入库 `2022.02` 目录的简单重命名。它保留了主回路 `40` 年寿命、`3mm 304` 不锈钢气箱、`IP67/IP43/IK07`、`1~6` 功能共箱、[[ALink]] 扩展以及 `Easergy P1`、`Easergy T300`、`PD110/TH110-R/CL110/SMD/PS100/PS50` 这套数字化架构，同时把 `Cr / Vr / M` 模块更明确地写进功能与参数页，并在部分 `12kV` 表格中出现了 `25kA` 短时耐受和 `63kA` 峰值耐受口径。

## Key Claims
- 末页文档号更新为 `ECATA1138 2025.02D0214`，说明这是 `2022.02D0214` 之后的目录修订版。
- 目录继续把 RM6 SeT 定位为 `24kV` 及以下气体绝缘开关设备，保留“鲁棒性设计、全能型架构、数字化”主叙事。
- 主回路 `40` 年寿命、寿命期内免维护、无需补气、`3mm 304` 不锈钢气箱、三工位负荷开关、安全阀和 `Digital logbook` 二维码等核心卖点继续保留。
- `1~6` 功能共箱、`NE / LE / RE / DE` 扩展方式和通过 [[ALink]] 侧扩且无需气体处理的实现方式继续保留。
- 与 `2022.02` 版相比，功能/模块说明页把 `Cr` 负荷开关母线柜、`Vr` 断路器母线柜和 `M` 计量柜更明确地列入目录结构，而不再只围绕 `C / F / V / D / PT` 展开。
- `M` 计量柜在新版中有独立模块位置，说明 `M` 不再只是“不能共箱”的约束性备注，而是被提升为更明确的选型对象。
- 新版仍给出 `PT` 和 `M` 柜只能单元式、不能共箱、只能通过可扩展 RM6 SeT 连接的边界，因此自由组合规则没有被放宽。
- 新版继续保留 `Easergy P1` 在核心元件和附件章节中的位置，说明保护路线仍是 `VIP40/45/400/410 + Easergy P1` 并行。
- 数字化章节继续覆盖 `Easergy T300`、`VT/LPVT`、`CT/LVCT`、`TH110-R`、`PD110`、`CL110`、`SMD` 和 `PS100/PS50`，说明 RM6 SeT 的 [[数字化中压运维]] 架构并未回退。
- 内部电弧栏仍给出基础型“未分类”与 `A FL / A FLR 20kA 1s` 口径，继续支持把 `A-FLR` 视为可选能力而不是默认全系标配。
- 新版目录中的部分表格开始出现 `20/25kA` 短时耐受、`50/63kA` 峰值耐受和相应时间参数组合，显示 RM6 SeT 至少在部分 `12kV` 柜型上新增了更高短路能力口径。
- 这些 `25kA / 63kA` 口径并不是整本目录对所有柜型的统一替换；当前更稳妥的理解是：新版为部分柜型补入了更高能力选项，而 `24kV` 侧仍保留 `20kA / 50kA` 主口径。
- 封面后的施耐德中国介绍页也发生修订，说明新版目录连带更新了企业概况页，但这部分属于宣传背景变化，不应覆盖具体产品技术参数。

## Contradictions
- 与 `2022.02` 的 [[RM6SeT]] 目录相比，新版更像“增补与细化”而不是技术路线反转：`40` 年寿命、`IP67/IP43/IK07`、`A-link`、`1~6` 功能共箱和数字化架构保持一致。
- 当前最需要保留的版本差异是：`2025.02` 明确加入 `Cr / Vr / M` 模块表达，并在部分 `12kV` 参数表中出现 `25kA / 63kA` 口径；因此不能再把 RM6 SeT 简化为只有 `C / F / V / D / PT` 且全系统一 `20kA / 50kA` 的单一版本。
