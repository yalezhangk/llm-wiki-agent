## Summary
这份 `48` 页中文 PDF 是 [[RM6S]] 与 [[RM6SPlus]] 的正式产品目录层，不是现有 `2021` [[RM6S]] 操作说明书的重复件，也不是 [[RM6SeT]] 的早期改名版本。它带有 `SCDOC1739 / 2018.10` 尾标，系统区分了 `630A` 侧扩的 `RM6-S` 与 `1250A` 顶扩的 `RM6-S Plus`，因此更适合被视为 RM6 路线中一条独立的 `12kV` 二次配电 GIS 分支资料链。

## Key Claims
- 封面写有 `RM6-S/RM6-S Plus 中压配电 12 kV 以下气体绝缘开关设备` 与 `产品目录`，PDF 元数据显示创建时间 `2018-10-15`、修改时间 `2018-10-15`。
- 末页尾标为 `SCDOC1739 / 2018.10`，说明它是正式目录层而不是操作手册、培训稿或演示稿。
- 公司页保留 `2017` 财年施耐德口径：`247 亿欧元`、全球 `14.2 万` 员工、中国 `17000+` 员工、`23` 家工厂、`8` 个物流中心、`9` 个分公司和 `37` 个办事处。
- 简介页把 `RM6-S/RM6-S Plus` 定义为 `12kV` 以下的中压开关柜，覆盖 `630A / 1250A`、`20kA / 25kA / 31.5kA`，面向公共配电、工业、基础设施和可再生能源场景。
- 文档强调高压导电部分位于不锈钢 `SF6` 气箱中，主回路 `30` 年寿命、寿命期内免维护且无需补气，工作连续性为 `LSC2A`。
- 运行条件页给出 `-5℃ ~ +40℃`、可选 `-25℃`、最大安装海拔 `1000m`、`+20℃` 时额定压力 `0.03 MPa`、柜体 `IK07`，并注明可选防浸式方案通过 `24 kV 50 Hz` 水下 `24` 小时试验。
- 内部电弧等级页给出 `RM6-S/RM6-S Plus` 的 `AFL 16/20/25/31.5kA 1s` 与 `AF/AFL` 计量单元边界，说明这条路线在 `2018` 已有多档内部电弧能力。
- `RM6-S` 侧重 `I / Q / B / O / IC / B/CB / M` 这组 `630A` 级功能体系，并保留 `O - 0.3s - CO - 180s - CO` 的快速自动重合闸顺序。
- `RM6-S Plus` 被明确定义为 `RM6-S` 的升级版：额定电流从 `630A` 提升到 `1250A`，柜间拼接从侧部扩展改为顶部扩展。
- `RM6-S Plus` 的主功能单元写为 `CB / B / G / CBBr / M / PT`，其中 `CB` 为 `1250A 31.5kA` 进线/出线真空断路器单元，`B` 为 `630A 25kA` 线路与变压器保护单元，`CBBr` 为 `1250A` 断路器母联单元。
- `RM6-S Plus` 目录明确给出顶部拼柜套管、普通电缆连接 `CB-C` 与固体母线连接 `CB-B` 两种电缆室方案，以及 `PT` 可位于母线任意位置的配置边界。
- 可扩展性页说明 `RM6-S` 通过侧部扩展口连接新增功能单元，且两台 `RM6-S` 柜之间的母线连接由特有 [[ALink]] 装置实现；`RM6-S Plus` 则通过顶部扩展真空连接和特定固体母线装置拼柜。
- 同一页还明确：`RM6-S` 扩展安装需预留大于 `450mm` 额外空间，而 `RM6-S Plus` 扩展无需额外安装空间。
- 远程控制页把远方控制等级正式写成 `0~4` 五级，并列出电话线调制解调器、无线电和 `GSM` 网络等通信实现。
- 附件与保护页说明 `RM6-S` 可装配 `MiCOM P114S`、`SEG WIC1` 等自供电保护继电器；`RM6-S Plus` 则以 `Sepam40` 作为 `CB` 进线柜标准配置、`Sepam20` 作为 `B` 出线柜标准配置。
