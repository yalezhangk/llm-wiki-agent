## Summary
[[空气绝缘开关设备]] 是当前 wiki 中与 [[气体绝缘开关柜]] 并列的中压开关柜路线，主要依靠空气作为主绝缘介质，并通过金属封闭、隔室分隔、联锁和 [[内部电弧防护]] 来满足安全与运行连续性要求。已有资料中，[[DNF7]]、[[MVnex]] 和 [[PIX]] 都属于这一大类，但电压等级、结构紧凑性和短路能力侧重点不同。

## Key Details
- [[DNF7]] 面向 40.5kV 户内交流 [[金属封闭开关设备]]，强调 LSC2B、PM 隔板和 IAC AFLR 31.5kA/1s。
- [[MVnex]] 面向 12kV [[金属铠装开关设备]]，标准型最高 4000A、40kA，并有 550/500 小型化分支。
- [[PIX]] 面向 12/17.5/24kV 高性能空气绝缘开关柜；英文目录给出 17.5kV、50kA、4000A 和 IAC AFLR 50kA/1s，中文目录补充 12kV/24kV、PIX-M 和 PIX-Gen。
- PIX V8 目录把产品族扩展为 7.2/12/24kV 口径，并以 550/650/800/1000/1200mm 等柜宽组合覆盖不同功能单元。
- PIX-Gen 是空气绝缘路线中的发电机出口断路器柜分支，中文目录给出 12kV、5000A/6300A 和 63kA 的高容量口径。
- [[PIX550]] 是 PIX 旗下 12kV 小型化分支，柜宽 550mm，额定电流 ≤1250A，短时耐受/开断能力 ≤31.5kA。
- [[PIX500]] 是 PIX 旗下 500mm 小型化空气绝缘分支，采用 HVX 固封式断路器和三工位开关，目录强调无 SF6。
- `SM6 Modular units Air insulated switchgear up to 36 kV` 正式目录把 [[SM6]] 的边界说得更清楚：它被定义为最高 `36 kV` 的空气绝缘开关柜，但 active parts integrated in an `SF6-filled, sealed for life` unit，并给出 `LSC2A`、`PI` 隔板与 `A-FL / A-FLR` 内部电弧分类。
- `SM6中压开关柜24kV产品目录` 则把这条路线进一步落到中国市场正式口径：适用 `24 kV` 以下二次变电站和工矿配电站，柜体尺寸收敛到 `375-750 mm` 宽、`1600 mm` 高、`1030 mm` 深，并把 `IP4X`、`LSC2A`、`PI` 和 `IAC A-FLR 20 kA/1s` 等工程口径写成中文目录参数。
- `SM6` 演示稿补入施耐德一条更偏环网柜的空气绝缘路线：封面写 `7.2~24kV` 空气绝缘环网柜，分类页把 [[SM6]] 与 [[RM6]] / [[FBX]]、[[Premset]] 并列。
- 但 [[SM6]] 并不是 [[PIX500]] 那种“无 `SF6`”的纯空气绝缘路线；其负荷/隔离开关柜和断路器柜页都明确写开关间隔充 `SF6` 作为绝缘介质，因此更适合理解为“空气绝缘柜体 + `SF6` 开关隔室”的边界型 RMU 平台。
- `AS12/AS24中压开关` 目录则补出另一种边界：它不是完整开关柜，而是施耐德提供给国内柜厂集成的 `12/24kV` 空气绝缘柜核心开关元件，覆盖 `IS/QS/DS/CS/ES` 功能分支。
- 这条 `AS12/AS24` 路线同样不是纯无气体绝缘：资料明确写三相旋转触头装在 `SF6 0.4 bar` 密闭压力气室内，年泄漏率 `<0.05%`，因此更适合理解为“空气绝缘柜体中的 `SF6` 开关核心”。
- `SC6中压开关元件产品目录` 则把这条元件路线推进到 `2017` 时点：文件名仍带 `AS12`，但正文题名已改为 [[SC6]]，并把参数与功能收敛到 `12kV`、`630A`、`20kA/4s` 和 `SIC/SQC`。
- `Smart SM6` 还把 `Easergy T300`、`TH110 / TH110-R / TH110-S`、`PD110` 和 `CL110` 接到 `SCADA / DMS` 与 Local HMI，说明空气绝缘路线同样可以接入 [[数字化中压运维]]。
- 这一路线常与 [[HVX]] 真空断路器、抽出式主开关设备、金属隔室和机械/电气联锁配合使用。
- 与 [[GMA]]、[[GMAirSeT]] 所代表的气体绝缘路线相比，空气绝缘开关设备通常体积更大，但维护和绝缘介质管理边界不同。
