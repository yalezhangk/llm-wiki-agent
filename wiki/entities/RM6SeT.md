## Summary
[[RM6SeT]] 是 [[施耐德电气]] `SeT` 系列中的 `24kV` 及以下 [[气体绝缘开关柜]] / 环网柜产品线，面向中压二次配电场景。当前 wiki 中它已经形成 `2022.02 -> 2025.02` 的正式目录版本链：两版都体现出相对早期 [[RM6]] 更工程化的一代口径，包括主回路 `40` 年寿命、`3mm 304` 不锈钢气箱、`1~6` 功能共箱、[[ALink]] 扩展、`Easergy P1` 保护，以及围绕 [[EcoStruxure]] 的完整数字化方案；而 `2025.02` 版又进一步把 `Cr / Vr / M` 模块和部分 `12kV 25kA / 63kA` 能力口径写得更明确。

## Key Details
- RM6 SeT 产品目录将其定义为 `24kV` 及以下气体绝缘开关设备，并强调“鲁棒性设计、全能型架构、数字化”。
- 主回路寿命期口径为 `40` 年以上，寿命期内免维护、无需补气，气箱采用 `3mm 304` 不锈钢。
- 基础绝缘与电流参数覆盖 `12kV / 24kV`、母线 `630A`、短时耐受 `20kA`、峰值耐受 `50kA`。
- 防护等级更完整：气箱 `IP67`，电缆室 `IP43`，机构室与低压间 `IP4X`，隔室间 `IP2X`，机械冲击等级 `IK07`。
- 功能体系覆盖 `C` 负荷开关、`F` 负荷开关-熔断器组合电器、`V` 真空断路器、`D` 提升柜和 `PT` 辅助电源/母线电压测量柜。
- `V` 功能真空断路器给出 `20kA` 开断能力、`M2 10000` 次机械寿命、`E2` 电寿命和自动重合闸顺序 `O - 0.3s - CO - 3 min - CO`。
- `F` 功能组合电器给出 `31.5kA` 开断能力，并把变压器保护熔断器方案纳入正式选型口径。
- 柜列支持 `1~6` 功能共箱，并支持 `NE / LE / RE / DE` 扩展方式；扩展通过 [[ALink]] 完成，无需气体处理。
- `PT` 和 `M` 柜不能共箱、只能单元式，是 RM6 SeT 自由组合的明确边界。
- 数字化能力覆盖 `Easergy T300`、`Easergy P1`、`VT/LPVT`、`CT/LVCT`、`TH110-R`、`PD110`、`CL110`、`PS100/PS50` 和 `SMD`。
- 文档还把 RM6 SeT 放入双电源自动切换、闭环配电网、自动重合闸和移动运维场景中。
- `2025.02` 修订目录延续了上述主骨架，但把 `Cr` 负荷开关母线柜、`Vr` 断路器母线柜和 `M` 计量柜更明确地纳入模块说明与参数页。
- `2025.02` 的部分参数表新增 `20/25kA` 短时耐受与 `50/63kA` 峰值耐受组合，说明 RM6 SeT 在部分 `12kV` 柜型上补入了更高短路能力选项；`24kV` 侧仍保持 `20kA / 50kA` 主口径。
