## Summary
这份两页中文折页的 PDF 元数据日期为 `2016-08-30`，末页尾标为 `SCDOC928-MV / 2016.09`，正文题名为 [[AS12AS24]] 路线中的 `AS12 中压开关 产品简介`。它不是更早 `AS12/AS24` 目录的同版副本，也不是后续 [[SC6]] 目录，而是一层居中的 `AS12` 单型号宣传/选型折页：参数收束到 `12kV 630A`，但仍保留 `IS / QS / CS / DS / ES` 全套功能分支、`375mm / 500mm` 宽度口径，以及 `AS12-ES` 隔离开关断路器组合。

## Key Claims
- 封面写明 `AS12 中压开关 产品简介`、`12kV 630A`、`可靠核“芯” 灵活便捷`，末页尾标为 `SCDOC928-MV`，时间口径为 `2016.09`。
- 资料仍将产品定位为柜厂集成用中压开关核心元件，而不是完整整柜；虽然源文件放在 `RM6` 目录下，但正文内容与 [[RM6]] 产品线无关。
- 第 1 页订购表覆盖 `AS12-IS / AS12-QS / AS12-CS / AS12-DS` 的 `375mm / 500mm` 宽度版本，并单列 `AS12-ES` 隔离开关断路器组合，注明相关图纸和安装说明需另查。
- 技术参数延续这条路线的核心边界：`12kV`、`630A`、工频耐压 `42/48kV`、雷电冲击耐压 `75/85kV`、短时耐受电流 `20kA/3s` 或 `25kA/1s`、峰值耐受 `50kA` 或 `63kA`。
- `AS12-ES` 页给出断路器组合口径：额定短路开断电流 `25kA`、隔离开关机械寿命 `5000/2000` 次、断路器机械寿命 `30000` 次、电气寿命 `E2` 级。
- 开关本体继续采用 `SF6` 密闭压力系统，正文写三相旋转触头装在相对压力 `0.4 bars` 的气室内，并强调符合 IEC “密闭压力系统”要求。
- 资料保留永磁旋弧灭弧技术叙事，强调电弧在永久磁铁电磁场作用下旋转、拉长并在电流过零时熄灭，以换取低操作过电压、低触头磨损和较长电气寿命。
- 安全边界继续围绕三工位开关、天然闭锁、观察窗、同轴机械指示、主开关/接地开关分离操作孔、后下方泄压和“五防”联锁展开。
- 操作机构层写出 `CIT / CI1 / CI2 / CS` 四类机构，覆盖手动、电动、储能、按钮分合与脱扣器等差异。
- 与 `2010` 的 [[AS12AS24]] 目录相比，这份折页已经收束到 `AS12 12kV`，但尚未像 `2017` 的 [[SC6]] 那样把功能进一步收敛到 `SIC / SQC`。

## Key Quotes
> “三工位开关，‘闭合’、‘断开’、‘接地’三位置天然闭锁” — 安全特征页

> “开关三相旋转触头被装入充满SF6气体、相对压力为0.4bars的气室内” — 密封性能页

## Contradictions
- 不与 [[AS12AS24]] 构成冲突，而是把更早 `12/24kV` 平台收束成 `2016` 的 `AS12 12kV` 单型号折页层，同时保留 `IS / QS / CS / DS / ES` 全套功能。
- 不应与 [[SC6]] 混为同版：`SC6` 目录已改写为 `SIC / SQC` 和 `20kA/4s` 收敛口径，而这份折页仍保留 `AS12` 命名、`IS / QS / CS / DS / ES` 和 `20kA/3s or 25kA/1s` 的旧平台表达。
- 虽然原文件位于 `raw/二次气体绝缘开关柜/RM6/`，但正文没有 `RM6` 产品线内容，因此不应把它归为 [[RM6]] 的版本来源。
