## Summary
这份 `SCDOC1544 / 2017.10` 中文目录是当前 [[FBX]] 资料链中更早的一版正式产品目录。它与后续 `2018.04` 和 `2020.05` 版共享 `24kV` 及以下、`630A / 20kA`、`FBX-C / FBX-E`、`LSC2A`、`SF6 0.03MPa` 和 `AFL/AF 16/20kA 1s` 等核心口径，但更突出可再生能源、风机塔筒内安装、`600mm` 门洞通过、`A-Link` 扩展和传统远控/故障隔离叙事，而尚未展开后续版本中的 [[EcoStruxure]] 在线监测章节。

## Key Claims
- [[FBX]] 被定义为 `24kV` 及以下、`630A`、`20kA` 的二次配电用中压开关装置，可通过 `T1` 熔断器或 `T2` 真空断路器实现变压器保护。
- 目录给出 `FBX-C` 紧凑型和 `FBX-E` 可扩展型两条柜型路线，并列出 `C`、`T1`、`T2`、`R`、`Sb` 和 `M` 功能单元。
- 简介页明确把 `FBX-E` 写入风机塔筒应用语境，说明其可凭紧凑尺寸通过狭窄门洞安装在风机塔筒内。
- 运行条件页给出 `-5℃ ~ +40℃` 常规温度、可选 `-25℃`、常规海拔 `1000m`、`SF6` 绝缘、`20℃` 额定压力 `0.03 MPa`、主回路 `IP67`、熔断器隔室 `IP65` 可选 `IP67`、低压隔室 `IP3X` 和柜体 `IK07`。
- 内部故障电弧等级给出 `C/T1/T2/R` 的 `AFL 16kA 或 20kA / 1s` 与 `M1~M4` 的 `AF 16kA 或 20kA / 1s`。
- 目录强调主回路 `30` 年使用寿命、寿命期内免维护、无需补气，以及 `LSC2A` 类工作连续性。
- `FBX-E` 通过 [[ALink]] 实现左右扩展，扩展时无需现场气体处理，并要求预留大于 `450mm` 的安装空间。
- 资料把 `Flair` 故障指示器、进线开关电动操作、远方故障隔离与供电恢复写入使用场景，说明这版更突出传统配网自动化和可再生能源运维叙事。
- 电缆选型页系统列出 `EUROMOLD`、`nkt`、`Südkabel`、`Tyco` 等连接件路线，并说明 `C / R` 功能单元支持每相双电缆连接。
- 末页给出 `SCDOC1544`、`2017.10` 和施耐德电气（中国）有限公司发布信息。
- 标准页中出现“`RM6-S开关柜满足以下标准和法规的要求`”字样，但上下文主体均为 FBX，宜视为模板/排版遗留，不能据此把该目录归到 [[RM6S]]。
