## Summary
这份 `SCDOC1273 / 2018.10` 中文资料不是单一产品目录，而是施耐德面向风电行业的系统级解决方案总册。它把风能场景拆成四组方案：风力发电机组系统、风场连接系统、并网电站系统，以及电力监控与视频安防系统，并把 [[FBX]]、[[FLUSARC]]、[[WSG]]、[[DNF7]]、[[Trihal]]、[[Premset]]、[[PLSCADA]] 和 [[iRIO]] 等产品与系统对象放进同一套风电叙事。与已有 `FBX`、`DVCAS` 等单产品资料相比，这份来源的价值不在补单柜细表，而在于补清施耐德 `2018` 时点对风机、风场连接、并网升压站和监控安防的一体化方案视角。

## Key Claims
- 资料将施耐德风能市场方案分成四类：风力发电机组系统方案、风场连接系统方案、并网电站系统方案、电力监控及视频安防系统方案。
- 风机系统章节强调中压设备需适应 `600mm` 宽塔筒门和 `2000m` 海拔机内安装语境，并把低压导电轨、低压配电柜、[[Premset]] 紧凑型中压开关柜、[[Trihal]] 升压变压器、接触器、空气断路器、PLC、HMI、UPS 和机柜组成一整套主电路与控制辅助系统。
- 其中 [[Premset]] 页给出 `12kV`、`630/1250A`、`20kA x 4s`、`25kA x 3s`、`25kA x 4s`、`31.5kA x 4s`、`375mm` 宽度和 `700mm` 前端对齐电缆连接等口径。
- 风场连接章节明确写到中压开关设备特别紧凑，宽度小于 `60cm`，可通过风机塔筒门，并列出预装式箱式变电站可选 `RM6`、`FBX`、`FLUSARC` 等不同型号开关设备。
- [[FLUSARC]] 被定位为 `40.5kV`、`SF6` 绝缘、真空断路器的环网开关设备，适用于系统标称电压最高至 `35kV` 的场内连接和风机变压器保护，并注明适用于海上及陆上型风机。
- [[FLUSARC]] 页给出典型组合 `C-C-C-CB`、`C-C-CB`、`CBL`，以及 `40.5kV`、`630A`、工频耐压 `95/118kV`、雷电冲击 `185/215kV`、`-5℃~+40℃` 可选 `-30℃` 等口径。
- 并网电站章节把 [[WSG]]、[[DNF7]]、[[HVX]] 以及 `FP 40.5kV SF6` 断路器并列为场站侧主要设备，并补出当时的组合口径：[[WSG]] `40.5kV / 2500A / 40kA / 4秒`、[[DNF7]] `40.5kV / 3150A / 31.5kA / 4秒`。
- 电力监控章节以 [[PLSCADA]] 为中心变电站综合自动化系统软件，以 [[iRIO]] 为风机侧智能终端，并配合 `ION7650`、`PowerLogic`、`MiCOM`、`Sepam`、`DLR`、`LOM` 和 `Pelco` 视频安防形成风场集中监控方案。
- [[iRIO]] 页给出中低压开关状态、柜门、告警、温湿度监测，`16` 个 `DI/DO`、`16` 个 `AI`、内置 Web 服务器、无线传输、email/SMS 报警等能力。
- 末页尾标为 `SCDOC1273 / 2018.10`，属于施耐德电气（中国）有限公司发布的系统级风能资料。

## Contradictions
- 与现有 [[FBX]]、[[DVCAS]]、[[WSG]]、[[DNF7]] 等来源不构成直接冲突；这份资料更像 `2018` 时点的行业解决方案总册，而不是单产品工程目录。
- 并网电站章节中的参数与各产品独立目录相比应视为方案宣传口径，后续工程选型仍应优先以对应产品正式目录或标准为准。
