## Summary
[[AS12AS24]] 是 [[施耐德电气]] 面向 `12/24kV` 二次配电空气绝缘开关柜场景提供的一组中压开关核心元件。当前已入库资料把它写成柜厂可集成的开关平台，而不是完整柜体产品线：它通过 `IS / QS / DS / CS / ES` 功能分支、`SF6` 密闭压力系统、永磁旋弧灭弧结构和一组保护/自动化附件，去支撑进出线、熔断器组合、双电源、PT 柜和隔离开关+断路器柜等不同功能；到 `2017` 的 [[SC6]] 目录中，这条路线又收敛为更聚焦 `12kV` 与 `SIC / SQC` 的后续版本。

## Key Details
- 已入库目录封面标题为 `中压开关 AS12/AS24 可靠核“芯” 灵活便捷`，PDF 元数据日期为 `2010-03-18`，修改时间为 `2010-11-24`。
- 产品定位不是整柜，而是“空气绝缘开关柜的核心元件”；资料明确写到柜体由国内开关柜制造商制造。
- 功能类型包括 `IS`、`QS`、`DS`、`CS`、`ES`，分别覆盖负荷开关柜、熔断器组合柜、双电源快速合闸柜、PT/中低压变压器柜和隔离开关+断路器柜。
- 额定参数主口径为 `12kV / 24kV`、`630A`；短时耐受电流常规为 `20kA/3s`，`AS12` 可选 `25kA/1s`。
- 开关本体采用 `SF6` 绝缘与灭弧，气室相对压力 `0.4 bar`，符合 IEC 密闭压力系统，资料写年泄漏率 `<0.05%`。
- 永磁旋弧灭弧是其技术核心，资料强调低操作过电压、触头磨损小和较长电气寿命；主开关机械寿命 `5000` 次，接地开关 `2000` 次。
- 安全设计包括三位置开关、天然联锁、同轴机械指示、观察窗、柜门联锁和事故泄压朝柜后释放。
- 操作机构分为 `CIT`、`CI1`、`CI2`、`CS` 等，覆盖手动/电动、快速分合闸、储能、分励脱扣和熔断器联动等差异。
- 可选附件体系较完整，包括 `Fusarc CF` 熔断器、`Amp 21D` 数字电流表、`Flair 22D` 故障指示器、`Talus 200I` 远程终端、钥匙闭锁和 `Sepam` 综合保护。
- `2017` 的 [[SC6]] 目录可视为这条平台的后续收敛资料：文件名仍带 `AS12`，但正文题名已改为 `SC6`，参数集中到 `12kV`、`630A`、`20kA/4s`，功能收敛为 `SIC / SQC`。
- 因此更适合把 [[AS12AS24]] 理解为施耐德早期“二次空气绝缘柜体 + SF6 开关核心 + 附件生态”的平台，而不是后续 [[SM6]]、[[RM6]]、[[FBX]] 那类整柜目录的简单前身。
