## Summary
[[GMA]] 是施耐德电气的中压一次 [[气体绝缘开关柜]] 产品线，资料将其描述为 SF6 气体绝缘单母线系统的一次配电设备。与 40.5kV 空气绝缘 [[DNF7]] 不同，GMA 主要覆盖 12/24kV 场景，强调紧凑、模块化、可扩展、密封气室和现场无需气体操作；与 [[GMAirSeT]] 相比，GMA 的关键差异是采用 SF6 而不是干燥空气/环保气体；与 [[GHA]] 相比，当前 GMA 资料更偏 12/24kV 紧凑一次配电口径，而 GHA 资料扩展到最高 40.5kV。

## Key Details
- 主要应用领域包括基础设施、政府机关、电子行业、交通行业和高端商建。
- 额定电压为 12/24kV，额定频率为 50Hz。
- 主母线额定电流不超过 2000A，分支母线额定电流不超过 1250A。
- 额定短时耐受电流为 31.5/25kA，额定短路持续时间为 4s。
- 断路器机械寿命为 20000 次，断路器等级为 M2-E2-C2，额定短路开断次数为 50 次。
- 典型柜型包括断路器柜、母联柜、隔离提升柜、隔离柜、直连柜和 PT 柜。
- 气室为符合 IEC 62271-1 和 GB/T 11022 的密封压力系统，每个充气隔室通过压力表监控，可选报警接点输出以实现远程监测。
- `2018.12` 的中文 `12kV` 目录层把 GMA 进一步收敛到主母线 `1250A`、分支母线 `630/1250A`、`25/31.5kA`、机械寿命 `10000` 次，并明确写出 `CB-C / CB-B / BC-CB / D / M / BB-PT` 柜型代码与 `420/510/600/750/800mm` 柜宽代码。
- 英文 2017 目录扩展了国际版参数视角：GMA 可覆盖 12/15/17.5/24kV、最高 2500A 和 31.5kA，并强调 450mm 宽馈线柜、IoT/Smart Grid ready、remote monitoring 和 [[EcoStruxure]] ready solutions。
- GMA 当前已经形成更清楚的版本链：`2017` 英文国际版偏 `24kV / 2500A` 与 IoT 叙事，`2018.12` 中文目录偏 `12kV / 1250A` 的中国工程目录层，`2023 V1.5` 中文目录则扩展为 `12/24kV`、主母线 `≤2000A` 的后续中国目录层。
