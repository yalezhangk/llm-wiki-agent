## Summary
这份中文 PDF 是 `2017-08-21` 元数据日期的正式 SM6 产品目录，封面标题为“中压开关柜 SM6 悉心守护 享受无忧 6～24kV 产品目录”，尾页文件号为 `SCDOC064-MV`，时间口径写为 `2017.08`。它不是现有 `2012` 英文目录和 `2019` 中文演示稿的重复件，而是面向中国市场的 `24kV` 正式目录版本：一方面把 [[SM6]] 的柜型、参数和订货体系落到中文工程口径，另一方面又继续保留“空气绝缘柜体 + `SF6` 密闭压力系统开关核心”的边界。

## Key Claims
- 封面标题写明 `SM6`、`6～24kV` 和“产品目录”，尾页文件号为 `SCDOC064-MV`，版本时间为 `2017.08`。
- PDF 元数据为 `CreationDate=2017-08-21`、`ModDate=2017-08-21`。
- 目录将 [[SM6]] 定义为可扩展模块组合式、金属密封开关柜系列产品，适用于工矿企业二次变电站 `24 kV` 以下的用户和配电站。
- 主要柜型覆盖 `IM / IMC / IMB / IMP` 负荷开关进出线柜、`QM / QMC` 熔断器负荷开关组合柜、`CRM` 带熔断器接触器柜、`DM1-A / DM1-D / DM1-S` 隔离开关加 `SF6` 断路器柜、`DMV` 隔离开关加真空断路器柜、`CM / CM2` 电压互感器柜、`GBC-A / GBC-B` 计量柜、`NSM-C` 双电源进线柜、`GBM` 母线升高柜、`GAM2` 进线电缆连接柜和 `TM` 站用变柜。
- 目录给出柜体尺寸口径：户内安装、`375 - 750 mm` 宽、`1600 mm` 高、`1030 mm` 深，电缆前接线。
- 额定参数口径覆盖 `7.2 / 12 / 24 kV`，额定电流 `630 / 1250 A`，短时耐受可选 `20 kA / 4s` 或 `25 kA / 1s`，短路关合电流写为 `50 kA / 63 kA`，`24 kV` 口径写为 `50 kA`。
- 内部电弧与结构等级写为 `LSC2A`、`PI`、`IP4X`，并明确给出内燃弧 `IAC: A-FLR, 20 kA / 1s`；另有基础等级 `12.5 kA 1 s, IAC: A-FL`。
- `QM / QMC` 最大分断能力写为 `12 kV 25 kA`、`24 kV 20 kA`；`DM1-A / DM1-D` 同样写为 `12 kV 25 kA`、`24 kV 20 kA`。
- 机械/电气寿命部分给出：`IM / IMC / IMB / IMP` 负荷开关 `5000` 次机械寿命；`DM1-A / DM1-D` 断路器 `10000` 次机械寿命，电气寿命按 `12.5 kA 40 次 / 20 kA 25 次 / 25 kA 20 次` 口径给出；`DMV` 则写为 `25 kA, 100 次开断`。
- 结构说明把柜体分成五个间隔：开关间隔、母线间隔、电缆间隔、操作机构间隔、控制保护间隔；其中开关与接地装置被密封在充满 `SF6` 的气室内，符合“密闭压力系统”标准要求。
- 断路器柜支持 `SFset` 与 `SF1` 两类断路器：`SFset` 配自能式保护装置和特殊传感器，不需要辅助电源；`SF1` 配综合保护继电器和标准传感器，需要辅助电源。
- 接触器柜支持 `Rollarc 400` 和 `Rollarc 400D` 两类接触器。
- 监控与保护层包括 `Sepam 20/40/80`、`VIP`、`Easergy T200 S`、自动转换设备、`Easergy Flair 22D` 故障指示器和 `Easergy Amp 21D` 安培表。
- `Easergy T200 S` 被写成“即插即用”多功能远程监测与控制接口，可采集开关位置、故障检测、电流值，向开关发送分合闸命令，并通过 `Modbus`、`DNP3.0 level 2`、`IEC 870-5-101` 与 `SCADA` 交换信息。
- `Flair 22D` 被列为故障电流指示器，`Amp 21D` 被列为自供电安培表，并写明 `SM6-24` 可在所有进线柜和熔断器柜上集成 `Amp 21D`。
- 目录还补入了应用方案页，如 `NSM-C` 双电源进线、开闭站方案、备用电源自动投切和 `S20 / S40` 保护器组合。

## Contradictions
- 与 `2012` 英文正式目录不构成冲突，而是更聚焦中国市场和 `24kV` 口径的后续版本：英文目录上探到 `36 kV` 并给出更宽柜型族，中文目录则收敛到 `24 kV` 以下并展开中文应用、选型、订货和自动化方案。
- 与 `2019` 中文演示稿不冲突，而是更正式的参数/订货底座：`2017` 目录强调工程参数、柜型和附件，`2019` 资料更偏市场叙事与 `Smart SM6` 在线监测层。
- 它再次确认 `SM6` 并不是纯无气体绝缘路线：虽然柜体按空气绝缘开关柜叙事，但开关与接地核心仍位于 `SF6` 密闭压力系统内。
